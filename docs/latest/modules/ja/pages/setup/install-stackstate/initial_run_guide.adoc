== デフォルトのユーザ名とパスワード

[tabs]
====
Kubernetes::
+
--
SUSE Observabilityは、デフォルトで次の管理者アカウントで構成されています：

* *ユーザー名：* `admin`
* *password:*インストール時に設定します。これは`generate_values.sh`スクリプトによって収集され、`values.yaml`にMD5ハッシュ形式で保存されます。
--

OpenShift::
+
--
SUSE Observabilityは、デフォルトで次の管理者アカウントで構成されています：

* *ユーザー名：* `admin`
* *password:*インストール時に設定します。これは`generate_values.sh`スクリプトによって収集され、`values.yaml`にMD5ハッシュ形式で保存されます。
--
====
