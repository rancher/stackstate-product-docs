= RancherとKubernetesの互換性

{stackstate-product-name} 2.6.3は、RKE2/Kubernetesを実行しているRancher管理のKubernetesクラスターでサポートされています。

以下のマトリックスは、互換性のあるRancherのバージョンとそれに対応するサポートされているRKE2/Kubernetesのバージョンを示しています。

|===
| Rancherのバージョン | RKE2 / Kubernetes Versions

| 2.11.x
| v1.30.11+rke2r1

| 2.12.x
| v1.30.11+rke2r1

| 2.13.x
a|
v1.30.11+rke2r1

v1.31.13+rke2r1

v1.32.10+rke2r1

|===

{stackstate-product-name}は、SUSEが提供するHelmチャートを使用してKubernetesまたはOpenShiftクラスターにインストールできます。インストールには Helm v3.x が必要で、チャートは以下のプラットフォームでサポートされています。

* *Kubernetes:*1.25から1.33
* *OpenShift:*4.14から4.19
