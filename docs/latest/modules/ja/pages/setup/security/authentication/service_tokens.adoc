== 概要

xref:use/security/k8s-service-tokens.adoc[サービストークン]を使用して、関連するユーザーアカウントなしでSUSE Observabilityで認証します。サービストークンは、CI/CDから認証するための推奨方法であり、エージェントやOpenTelemetryからテレメトリデータを送信する際にも使用されます。サービストークンの作成と使用方法は、xref:use/security/k8s-service-tokens.adoc[サービストークンのドキュメント]に記載されています。
