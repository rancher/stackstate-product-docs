= 保存されたデータを削除する
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability 自己ホスト型

SUSE Observability のデータは、4つの異なるセットに分かれています：

* Elasticsearch データ
* Kafka トピックデータ
* StackGraph データ
* メトリクスデータ

これだけのデータを保存するためには、管理手段を持つことが重要です。SUSE Observability では、標準で30日間のデータ保持期間が設定されています。これは、SUSE Observability CLI を使用してニーズに応じて設定できます。xref:/setup/data-management/data_retention.adoc[SUSE Observability データ保持についての詳細を確認する]。

== データを手動で削除する

Kubernetes 上で実行されている SUSE Observability の保存データを削除するには、クリーンインストールを実行することをお勧めします：

. xref:/setup/install-stackstate/kubernetes_openshift/uninstall.adoc#_un_install_the_helm_chart[SUSE Observability をアンインストールする]
. xref:/setup/install-stackstate/kubernetes_openshift/uninstall.adoc#_remove_remaining_resources[すべての PVC を削除する]
. 以前と同じ設定を使用して、xref:/setup/install-stackstate/kubernetes_openshift/kubernetes_install.adoc#_deploy_suse_observability_with_helm[Kubernetes] または xref:/setup/install-stackstate/kubernetes_openshift/openshift_install.adoc#_deploy_suse_observability_with_helm[OpenShift] に再度 SUSE Observability をインストールします。
