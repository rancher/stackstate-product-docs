= README
:description: SUSE Observability

OpenTelemetryはhttps://opentelemetry.io/docs/languages/[多くの言語向けのSDK]を提供し、アプリケーションを(自動)インストゥルメンテーションします。インストゥルメンテーションにより、アプリケーションはSUSE Observabilityにトレース、メトリクス、ログを送信できます。自動インストゥルメンテーションは、SDKをアプリケーションに含める以外、手動での操作を必要とせず、人気のあるライブラリやフレームワークのトレース、メトリクス、ログを取得します。

OpenTelemetryには、ここに文書化されている以上に多くのSDKがあります。ここに文書化されていない言語を使用している場合は、https://opentelemetry.io/docs/languages/[opentelemetry.io]のドキュメントと、SUSE Observabilityに必要なxref:/setup/otel/instrumentation/sdk-exporter-config.adoc[SDKエクスポーターの設定]を組み合わせて使用できます。
