== 特定のネットワークプロトコルのトレースをプロセスエージェントで無効にします。

プロセスエージェントで特定のネットワークプロトコルのトレースを無効にするには、Helmチャートの設定`processAgent.disabledProtocols`を構成します。以下の例では、`http2`および`postgres`プロトコルを無効にします：

----
--set "processAgent.disabledProtocols[0]=http2" \
--set "processAgent.disabledProtocols[1]=postgres"
----

== プロセスエージェントを無効にします。

プロセスエージェントがあなたの環境でサポートされていない場合は、Helmチャートの設定`nodeAgent.containers.processAgent.enabled`で無効にします。

----
--set nodeAgent.containers.processAgent.enabled=false
----
