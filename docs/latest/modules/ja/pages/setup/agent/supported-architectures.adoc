= SUSE Observability Agentがサポートするアーキテクチャ
:revdate: 2025-07-10
:page-revdate: {revdate}

SUSE Observabilityは、特定のハードウェアアーキテクチャで動作するように設計され、テストされています。インストールと運用は、以下のシステムでサポートされています：

* `x86_64`（AMD64またはIntel 64とも呼ばれます）
* `arm64`（AArch64とも呼ばれます）

デプロイメントを進める前に、ターゲット環境がこれらの要件を満たしていることを確認してください。他のアーキテクチャでSUSE Observabilityをインストールまたは実行することはサポートされていません。
