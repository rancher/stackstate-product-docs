= キーボードのショートカット
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

以下のキーボードショートカットを使用して、SUSE Observability UIをナビゲートできます。

|===
| キーボードショートカット | アクション

| `CTRL` + `SHIFT` + `F`
| xref:/use/views/k8s-topology-perspective.adoc[トポロジーの視点]で名前でコンポーネントを検索します。

| `CTRL` + `S`
| ビューの以前に保存された設定を上書きします。これにより、xref:/use/views/k8s-filters.adoc[ビューのフィルター]やxref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[視覚化設定]に加えた変更が含まれるように保存されたビューを更新します。

| `CTRL` + `SHIFT` + `S`
| 現在のビュー設定を新しいビューとして保存します。新しいビューには、現在適用されているすべてのxref:/use/views/k8s-filters.adoc[ビューのフィルター]とxref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[視覚化設定]が含まれます。

| `CTRL` + `SHIFT` + `R`
| 保存されていない変更をリセットし、元のビュー、サブビュー、または探索モードに戻ります。これにより、xref:/use/views/k8s-filters.adoc[ビューのフィルター]やxref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[視覚化設定]に加えたすべての保存されていない変更がリセットされます。

| `SHIFT` + `1`
| ビュー、サブビュー、または探索モードで、xref:/use/views/k8s-topology-perspective.adoc[トポロジーの視点]を開きます。

| `SHIFT` + `2`
| ビュー、サブビュー、または探索モードで、xref:/use/views/k8s-events-perspective.adoc[イベントの視点]を開きます。

| `SHIFT` + `3`
| ビュー、サブビュー、または探索モードで、xref:/use/views/k8s-metrics-perspective.adoc[メトリクスの視点]を開きます。

| `SHIFT` + `4`
| ビュー、サブビュー、または探索モードで、xref:/use/views/k8s-traces-perspective.adoc[トレースの視点]を開きます。
|===
