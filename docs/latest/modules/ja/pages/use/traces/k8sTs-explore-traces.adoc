== 痕跡の表示

image::k8s/k8s-pod-view-menu.png[ポッドメニューの痕跡]

ポッド、サービス、デプロイメント、ステートフルセット、またはデーモンセットに関連する痕跡を表示するには、「痕跡」メニューをクリックしてください。
これにより、xref:/use/views/k8s-traces-perspective.adoc[痕跡の視点]に移動します。
