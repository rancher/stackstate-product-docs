= イベントの視点
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

== 概要

イベントの視点は、現在のxref:/use/views/k8s-view-structure.adoc[ビュー]またはフィルタリングされたトポロジー内の要素に対するイベントと変更を表示します。

image::k8s/k8s-events-perspective.png[イベントの視点]

== 表示されるイベントをフィルタリング

表示されるイベントは、ビューUIの右上隅にある*フィルター*メニューでフィルタリングできます。イベントフィルターは、ビューに表示されるすべてのイベントに適用されます：イベントの視点、ハイライトの視点、およびタイムライン。

=== プロパティでフィルタリング

xref:/use/views/k8s-filters.adoc#_filter_events[イベントフィルター]は、*フィルター*メニューで設定され、イベントのカテゴリ、タイプ、ソースシステム、およびタグに基づいて表示されるイベントを絞り込むために使用できます。

=== ソースコンポーネントでフィルタリング

xref:/use/views/k8s-filters.adoc#_filter_topology[トポロジーフィルター]は、*フィルター*メニューで設定され、表示されるイベントの要素（コンポーネントと関係）を定義します。適用されたトポロジーフィルターまたはビュー自体に一致する要素に関連するイベントのみが表示されます。トポロジーフィルターを更新することで、表示されるイベントのコンポーネントを調整できます。

=== タイムスタンプでフィルタリング

イベントの視点とイベントリストは、ビューUIの下部にあるタイムラインで選択されたxref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_telemetry_interval[テレメトリ間隔]に一致するイベントを表示します。テレメトリ間隔を調整して、その時点で生成されたイベントのみを表示します。

== タイムトラベル

イベントの視点は、生成されたイベントをリストします：

* xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_topology_time[トポロジー時間]で指定された時点に存在したトポロジー要素によって。
* 選択されたxref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_telemetry_interval[テレメトリ間隔]内で

これにより、特定の歴史的な時点で利用可能なトポロジー要素によって生成されたイベントと、特定の時間ウィンドウ内で生成されたイベントの両方にタイムトラベルできます。新しいトポロジー時間を選択すると、その歴史的な時点で利用可能だったトポロジーにタイムトラベルします。イベントリストは、その時点で利用可能なトポロジー要素と選択したテレメトリ間隔内のイベントを含むように更新されます。

次に例を示します。

* *テレメトリ間隔*を調整して、表示されるイベントの数を増減させてください。
* *トポロジー時間*を調整して、その歴史的な時点で利用可能なトポロジーにタイムトラベルしてください。トポロジー時間に存在したトポロジー要素によって、選択したテレメトリ間隔内で生成されたイベントが表示されます。
* イベントの*プロパティ*からタイムスタンプをクリックして、このトポロジー時間にジャンプします。これにより、生成されたイベントを表示するためにイベントリストが更新されます：
 ** その瞬間に存在したトポロジー要素によって。
 ** 指定されたテレメトリ間隔内で（必要に応じて、選択したタイムスタンプに合わせて調整されます）。
