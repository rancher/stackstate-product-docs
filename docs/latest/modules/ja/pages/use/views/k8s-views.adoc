== 概要

SUSE ObservabilityはKubernetesとそのコンポーネントに関する深い知識を持っています。クラスターにSUSE Observabilityエージェントをインストールした後、Kubernetesアプリケーションのトポロジーを自動的に検出し、可視化します。これには、デプロイメント、ポッド、サービス、イングレスなど、アプリケーションを構成するKubernetesリソースが含まれます。また、クラスターを構成するKubernetesインフラストラクチャのトポロジーも自動的に検出し、可視化します。これには、ノード、ネームスペース、永続ボリュームが含まれます。

SUSE Observabilityは、以下のKubernetesネイティブリソースのための専用の概要およびハイライトページを提供しています：

image::k8s/k8s-menu.png[Kubernetesのビュー]

他のすべてのKubernetesリソースは、トポロジービューで認識され、可視化されます。

== 概要ページ

概要ページは、環境内の特定のKubernetesリソースの高レベルな概要を提供します。これらの概要は、リソースに関する最も重要な情報を強調表示します。これには、健康状態、年齢、ネームスペース、クラスターなどが含まれます。

image::k8s/k8s-service-overview.png[概要ページ]

概要ページから、Kubernetesリソースの1つをクリックして、そのリソースのハイライトページに移動できます。

== ハイライトページ

ハイライトページは、特定のKubernetesリソースに関する最も重要な情報の概要を示します。ここでは、リソースの健康状態、リソースでアクティブなモニター、およびリソースで発生したイベントを確認できます。また、リソースの主要なメトリクスも表示します。

image::k8s/k8s-pod-highlights.png[ハイライトページ]
