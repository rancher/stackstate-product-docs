= メトリクスの視点
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

メトリクスの視点は、選択されたリソースのメトリクスを表示します。

image::k8s/k8s-metrics-perspective.png[メトリクスの視点]

== グラフ

チャートは、選択されたコンポーネントのメトリクスデータをほぼリアルタイムで表示します - データは30秒ごとに取得されます。処理が停止してデータの受信がなくなると、データが少なくとも30秒ごとに左へシフトする中で、その処理は最終的にチャートから消えます。30秒の間隔中にさらにデータが到着した場合、それはチャートにプッシュされます。
