== 概要

SUSE Observabilityのビューでは、IT環境の一部を監視および確認することができます。ビューの構造は、その部分（ビュー）のデータを効率的にフィルタリングおよび視覚化するように調整されています。

=== フィルタ

ビューUIの右上隅にある*フィルター*メニューを使用すると、ビューに表示されるコンポーネント（トポロジー）、イベント、およびトレースをフィルタリングできます。適用されると、フィルターはビュー内のすべての視点の内容に影響を与えます。

* xref:/use/views/k8s-filters.adoc[フィルター] - ビュー内のコンポーネント（トポロジー）、イベント、およびトレースをフィルタリングします

=== パースペクティブ

ビューの*視点*は、ビューUIの左上隅にタブとして表示され、異なるレンズを通してビュー内のすべてのデータを視覚化することができます：

* xref:/use/views/k8s-overview-perspective.adoc[概要視点]またはxref:/use/views/k8s-highlights-perspective.adoc[ハイライト視点] - あなたがいるビューのタイプに応じて
* xref:/use/views/k8s-topology-perspective.adoc[トポロジー視点] - ビューコンポーネントの依存関係マップ
* xref:/use/views/k8s-events-perspective.adoc[イベント視点] - トポロジー上で発生しているすべてのイベント
* xref:/use/views/k8s-metrics-perspective.adoc[メトリクス視点] - 最も関連性の高いコンポーネントの主要なメトリクス
* xref:/use/views/k8s-traces-perspective.adoc[トレース視点] - トポロジー上で実行されているトレース情報

[NOTE]
====
*すべての視点*は、xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc[タイムライン]の設定に基づいて内容を更新します。
====
