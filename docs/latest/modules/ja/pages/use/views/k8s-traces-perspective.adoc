= トレースの視点
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

トレースの視点は、コンポーネントとその関連メトリクスのスパンを表示します。
これにより、SUSE Observability で IT インフラストラクチャ内のアプリケーションのパフォーマンスを直接監視できます。

リスト内の任意のスパンをクリックすると、それに属する子孫スパンが表示されます。
開くと、スパンのタイムラインは各子孫が開始および完了した時刻を示します。
同じトレース内の親スパンはデフォルトで折りたたまれていますが、必要に応じて開くことができます。 +
スパンは、OpenTelemetry の ServiceName に応じて異なる色で表示されます。
トレースを検査し、そのスパンのリストを見るとき、任意のスパンをクリックしてさらに詳細を確認できます。

image::k8s/k8s-traces-perspective.png[トレースの視点]

== トレースフィルタ

トレースフィルタは、スパンの状態（エラー、正常、または未設定）、親の種類（外部、内部、またはルート）、および期間に基づいて表示されるトレースを絞り込むことを可能にします。

これらのフィルタに加えて、トレースは SUSE Observability UI の下部にあるタイムラインコントロールで選択された *時間ウィンドウ* に一致します。
時間ウィンドウを調整して、その時間のトレースのみを表示します。

=== 期間フィルタ

2 番目のスクリーンショットに示されているように、ヒストグラムをブラッシングすることでスパンの期間インターバルを選択できます。
ズームアウトは、すべてのフィルタをクリアするか、全範囲をブラッシングすることで実現できます。

== スパンの詳細

SUSE Observability では、xref:/use/views/k8s-view-structure.adoc[ビュー] が特定の Kubernetes リソースに関連する IT インフラストラクチャの一部を表示します。
トレースの視点は、リソースに関連するスパンとその子孫を表示します。
子孫スパンは別のリソースから発生する可能性があるため、スパンの詳細からそのスパンへ移動することが可能です。
「親スパン」リンクをクリックすると、ビューが展開されて完全なトレースが表示されます。

image::k8s/k8s-traces-perspective-span-details.png[子孫スパンの詳細]

上記の2つの画像は、カートから注文を作成することが主な責任であるチェックアウトサービスを示すことで、これらの概念を説明しています。
注文を出すリクエストのトレースとそのスパンの例を見ることができます。
子孫スパンが強調表示され、その詳細が表示され、関連するコンポーネントへのリンクが含まれています。

同様に、トレースをブラッシングすると、トレースの特定の時間セクションにズームインします。
選択は後でクリックすることでリセットできます。

== タイムトラベル

トレースの視点を使用する際には、他の視点と同様に、ライブモードまたはxref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_time_travel[過去にタイムトラベルする]ことができます。
