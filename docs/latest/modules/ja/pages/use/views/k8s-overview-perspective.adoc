= 概要ビュー
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

概要ビューでは、ビュー内のすべてのコンポーネントのリストが表示されます。ビューの種類やビュー内のコンポーネントによって、概要ビューの構造は異なります。

例えば、xref:/use/views/k8s-views.adoc[kubernetes views]で使用されるテーブル構造は、各ビューに含まれるコンポーネントタイプの最も重要なプロパティを反映します。以下のように、`services`ビューは`pods`ビューとは異なるテーブル構造を持っています。

image::k8s/k8s-overview-perspective-table-comparison.png[概要テーブル構造の比較]

xref:/use/views/k8s-custom-views.adoc[カスタムビュー]とxref:/use/views/k8s-explore-views.adoc[探索ビュー]の場合、概要ビューのテーブルは、ビューに含まれる可能性のあるコンポーネントタイプの多様性のため、最も一般的なプロパティで構成された汎用的な一律の構造を持ちます。

image::k8s/k8s-overview-perspective-generic-table.png[概要テーブルの汎用構造]

概要ビューのほとんどのビュータイプではテーブルレイアウトが使用されますが、場合によってはカードレイアウトも提供され、概要ビューの内容を表示する異なるモードに切り替えることができます。

image::k8s/k8s-overview-perspective-cards-layout.png[概要カードレイアウト]
