= 通知
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

SUSE Observability における通知は、SUSE Observability の xref:/use/alerting/k8s-monitors.adoc[監視] が発火した際に、サードパーティアプリケーションに通知する方法です。通知は、インシデント管理ツール（PagerDuty や Opsgenie など）、チケットシステム（JIRA や ServiceNow など）、または他のコラボレーションプラットフォーム（Slack など）と統合されます。他のツールでは、通知をアラートと呼ぶことがあります。

通知は、SUSE Observability コンポーネント上の監視の健康状態の変化によってトリガーされます。設定された通知チャネルを通じて、健康、コンポーネント、および監視情報がこれらの外部システムに送信されます。
