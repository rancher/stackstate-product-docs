== Teamsの通知を設定する

Slackに通知を送信するには、以下の手順に従ってください：

. <<_create_a_power_automate_flow,Power Automateフローを作成する>>
. <<_add_and_test_the_channel,チャネルを追加してテストする>>

=== Power Automateフローを作成する

Teamsで「Webhook」テンプレートから新しいフローを作成します。

image::k8s/notifications-teams-webhook-template.png[Webhookテンプレートからフローを作成する]

通知が投稿されるチームとチャンネルを選択し、フローを保存します。

フローを編集し、「TeamsのWebhookリクエストが受信されたとき」ボックスをクリックします。
HTTP URLパラメータをコピーします。

image::k8s/notifications-teams-select-url.png[フローからURLを選択する]

== チャネルを追加してテストする

image::k8s/configure-teams-channel.png[Teamsチャンネルを設定する]

SUSE Observabilityに戻ると、Webhook URLを使用して通知チャンネルを作成できます。

== 通知用のTeamsメッセージ

通知が開かれたり閉じられたりすると、チャンネルに新しいTeamsメッセージが作成されます。
