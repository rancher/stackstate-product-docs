== Slack通知を設定する

Slackに通知を送信するには、以下の手順に従ってください：

. <<_connect_slack_workspace,SlackワークスペースにSUSE Observabilityを接続する>>
. <<_select_a_slack_channel,通知を送信するSlackチャネルを選択する>>
. <<_add_and_test_the_channel,チャネルを追加してテストする>>

=== Slackワークスペースを接続する

「ワークスペースを選択」ボタンをクリックします。これにより、SUSE Observabilityがチャネルをリストし、Slackチャネルにメッセージを送信するための権限を求めるSlackのウェブページが開きます。右上隅に希望のSlackワークスペースが選択されていることを確認してください。

続行するには「許可」をクリックします。

=== Slackチャネルを選択する

image::k8s/notifications-slack-channel-configuration.png[Slackチャネルを選択する]

SUSE Observabilityに戻ると、リストからSlackチャネルを選択できるようになります。通知メッセージを送信するチャネルを選択してください。

[NOTE]
====
プライベートチャネルは自動的にはリストに表示されません。プライベートチャネルを選択するには、まずSUSE Observabilityボットをプライベートチャネルに招待する必要があります。チャネル内でボット`@SUSE Observability`に言及するSlackメッセージを送信し、「招待する」を選択してください。これで、チャネルがリストに表示されるようになります。
====


== チャネルを追加してテストする

最後に「チャネルを追加」ボタンをクリックします。これにより、右側のチャネルリストにチャネルが追加されます。「テスト」ボタンが表示されます。それを押して、選択したSlackチャネルにテストメッセージが表示されることを確認してください。

== 通知用のSlackメッセージ

通知が開かれると、新しいSlackメッセージが作成されます。このメッセージは、通常、健康状態が変化したときにのみ更新されます。メッセージのスレッドでは、すべての変更がメッセージとして表示されます。最後に、通知が閉じられると、Slackメッセージが再度更新され（健康状態を示すバーは灰色になります）、通知が閉じられた理由を説明する最終メッセージがSlackスレッドに追加されます。
