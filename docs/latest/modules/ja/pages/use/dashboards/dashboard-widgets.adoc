= ダッシュボードウィジェット
:revdate: 2025-11-24
:page-revdate: {revdate}
:description: リソースからデータをさまざまなウィジェットで可視化します。

ウィジェットはダッシュボードの基本要素です。それらは通常、1つまたは複数のKubernetesリソースからのデータを可視化します。ウィジェットは、ダッシュボードの視聴者に静的なテキスト情報を表示することもできます。

== 新しいウィジェットを追加する

新しいウィジェットは、次の2つの方法のいずれかで追加できます：

* ダッシュボードの右上にある*ウィジェットを追加*ボタンをクリックします。

* 空のダッシュボードでは、画面の中央にある*ウィジェットを追加*セクションをクリックします。新しいウィジェットを設定できるドロワーが開きます。

image::dashboards/add-a-widget.png[ウィジェットを追加するさまざまな方法]

== 既存のウィジェットを追加する

{stackstate-product-name}の個々のコンポーネントを表示しているときに、"その他のアクション"メニューを使用してダッシュボードにチャートを直接追加することができます。

image::dashboards/add-existing-widget.png[既存のウィジェットを追加する]

これは、個々のウィジェットの同じ"その他のアクション"メニューを介して、すべてのピン留めされたアイテムにも利用可能です。

さらに、ピン留めされたアイテムのドロワー内の"その他のアクション"を使用するか、ドロワーの下部にある"すべてのアイテムをダッシュボードに追加"ボタンをクリックすることで、すべてのピン留めされたアイテムをダッシュボードに追加できます。

image::dashboards/add-all-pinned-items.png[すべてのピン留めされたアイテムを追加]

ダッシュボード内の結果として得られるウィジェットには、追加元のリソースへのリンクがあります。

== ウィジェットの設定

各ウィジェットには、いくつかの設定オプションがあります。いくつかはすべてのウィジェットに共通であり、他はウィジェットのタイプに基づいています。

ウィジェットの設定を編集するには、「その他のアクション」メニューを使用し、「ウィジェットを編集」を選択します。

image::dashboards/configure-a-widget.png[すべてのピン留めされたアイテムを追加]

=== 一般的なウィジェットフィールド

|===
| フィールド名 | 説明

| 名前
| ウィジェットのヘッダーに表示される短く説明的な名前。

| 説明
| ウィジェットのヘッダーに、ツールチップで表示される追加の詳細。

| タイプ
| <<types,視覚化の種類>>。
|===

これらのフィールドに加えて、すべてのウィジェットに共通のタブがあります：

|===
| タブ名 | 説明

| リンク
| ウィジェットの名前の下に表示される関連リソースリンク。最初のリンクは常に表示されます。他のリンクはドロップダウンメニューで利用可能です。

| YAML
| ウィジェットの生のYAML設定で、表示および編集が可能です。
|===


=== クエリタブ

リソースからデータを表示することを目的としたウィジェットタイプには、*クエリ*タブがあります。ウィジェットのデータを取得するために、1つ以上のPromQLクエリを追加できます。


=== 種類 [[types]]

==== TimeSeriesChart

`TimeSeriesChart`は、折れ線グラフで時系列データを視覚化します。ウィジェット設定内の各クエリは、グラフ上に1本の線を生成します。

image::dashboards/time-series-chart-example.png[例の時系列チャート]

次の設定を行うことができます：

* *凡例* - 凡例の位置とモード。
* **視覚** - null 値を接続して連続した線を作成します。
* **Y軸** - Y軸の単位、ラベル、最小/最大値。
* **閾値** - 重要な値の閾値を示すために、チャートに水平線を追加します。

==== 棒グラフ

`BarChart` は、棒グラフで時系列データを視覚化します。それは `TimeSeriesChart` と同じ設定の凡例、Y軸、閾値を持っています。

image::dashboards/bar-chart-example.png[例の棒グラフ]

==== 統計チャート

`StatChart` は、スパークラインチャートと一目で読みやすい単一の値を表示します。閾値は、値に基づいてテキストの色を変更するように設定できます。

image::dashboards/gauge-chart-example.png[例のゲージチャート]

次の設定を行うことができます：

* **スパークライン** - スパークラインチャートを表示または非表示にします。
* **単位** - 値の単位。
* **小数点** - 表示する小数点以下の桁数。
* **計算** - 時系列データから値を計算するために使用される方法。

==== GaugeChart

`GaugeChart` は、中央に値を持つゲージを表示します。値が閾値を超えると、ゲージとテキストの色が変わります。

image::dashboards/stat-chart-example.png[例の統計チャート]

次の設定を行うことができます：

* **単位** - 値の単位。
* **小数点** - 表示する小数点以下の桁数。
* **最大値** - ゲージの最大値です。
* **計算** - 時系列データから値を計算するために使用される方法。

==== マークダウン

`Markdown`ウィジェットはマークダウンテキストを表示します。これは、ダッシュボードに説明やその他の情報を追加するために使用できます。

image::dashboards/markdown-chart-example.png[例のマークダウンチャート]
