== ウィジェットを配置し、サイズを変更する

ダッシュボード上のウィジェットを整理して、自分に合ったレイアウトを作成できます。

* ウィジェットを移動するには、ウィジェットのタイトルエリアをクリックしてドラッグし、新しい位置に移動します。
* ウィジェットのサイズを変更するには、ウィジェットの右下隅をクリックしてドラッグします。

ウィジェットを移動またはサイズ変更すると、ダッシュボード上の他のウィジェットは自動的に再配置されます。

== ウィジェットを編集または削除する

. ウィジェットを編集または削除するには、ウィジェットの右上にある「その他のアクション」アイコンをクリックします。

. *ウィジェットを編集する* または *ウィジェットを削除する*を選択します。

ウィジェットのデータをより詳細に確認するには、メトリックインスペクタードロワーの「その他のアクション」ボタンの横にあるアイコンをクリックします。
