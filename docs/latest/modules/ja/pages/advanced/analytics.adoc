= 分析ページ
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability 自己ホスト型

分析ページは、xref:/develop/reference/k8sTs-stql_reference.adoc[STQLクエリ]を使用して、監視データを照会および分析するための強力で柔軟なインターフェースを提供します。
このインタラクティブな環境は、事前に構築されたダッシュボードを超えて、動的環境の健康、パフォーマンス、および依存関係に関するより深い洞察を得ることを可能にします。

[CAUTION]
====
*分析ページは廃止することになっていますが、特定の権限を設定することで引き続き有効にできます。ただし、StackPacksを活用して、より強力で積極的にサポートされている監視および分析の代替手段を提供することを強くお勧めします。*

分析ページを有効にするには、2つの権限が必要です：

* `access-analytics`
* `execute-scripts`

詳細情報：xref:/setup/security/rbac/rbac_permissions.adoc[権限管理]

====
