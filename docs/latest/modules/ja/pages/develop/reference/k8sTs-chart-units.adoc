= チャートでサポートされているユニット
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

このページには、チャートでサポートされているすべてのユニットが含まれています。各ユニットについて、最初にユニットの説明が記載され、その後にSUSE Observability APIで使用すべきユニット名が続きます。例えば、「科学的表記」で表示される値を取得するには、メトリックバインディングでユニットの値として`sci`を使用します。

ユニットのレンダリングは、値のサイズに自動的に調整されます。例えば、チャートのユニットが`bytes`の場合、値は`bytes(IEC)`として解釈されます。値`100`は`100B`としてレンダリングされますが、値`5242880` (`5*1024*1024`)は`5MiB`として表示されます。
