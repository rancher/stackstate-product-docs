= 图表支持的单元
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

此页面包含图表支持的所有单元。每个单元首先会提到单元的描述，然后是应在 SUSE Observability API 中使用的单元名称。例如，要以“科学计数法”显示值，请使用`sci`作为度量绑定中单元的值。

单元的渲染会自动根据值的大小进行调整。例如，如果图表的单元是`bytes`，则值被解释为`bytes(IEC)`。值为`100`时将渲染为`100B`，但值为`5242880` (`5*1024*1024`时将显示为`5MiB`。
