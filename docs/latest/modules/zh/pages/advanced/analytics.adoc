= 分析页面

:revdate: 2025-07-10

:description: SUSE Observability 自托管

分析页面提供了一个强大而灵活的界面，可以使用xref:/develop/reference/k8sTs-stql_reference.adoc[STQL查询]查询和分析您的可观测性数据。
这个交互式环境使您能够超越预构建的仪表板，深入了解动态环境的健康、性能和依赖关系。

[CAUTION]
====
*分析页面已弃用，但您仍然可以通过配置特定权限来启用它。然而，我们强烈建议利用StackPacks来增强可观测性和分析，因为它们提供了更强大且得到积极支持的替代方案。*

启用分析页面需要两个权限：

* `access-analytics`
* `execute-scripts`

更多详细信息：xref:/setup/security/rbac/rbac_permissions.adoc[权限管理]

====
