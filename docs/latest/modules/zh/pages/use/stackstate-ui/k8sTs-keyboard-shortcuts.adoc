= 键盘快捷键
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

下面的键盘快捷方式可用于导航 SUSE Observability 用户界面。

|===
| 键盘快捷方式 | 操作

| `CTRL` + `SHIFT` + `F`
| 在 xref:/use/views/k8s-topology-perspective.adoc[拓扑视角] 中按名称搜索组件。

| `CTRL` + `S`
| 覆盖先前保存的视图设置。这将更新保存的视图，以包括对 xref:/use/views/k8s-filters.adoc[视图过滤器] 或 xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[可视化设置] 所做的任何更改。

| `CTRL` + `SHIFT` + `S`
| 将当前视图设置保存为新视图。新视图将包括所有当前应用的 xref:/use/views/k8s-filters.adoc[视图过滤器] 和 xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[可视化设置]。

| `CTRL` + `SHIFT` + `R`
| 重置任何未保存的更改并恢复到原始视图、子视图或探索模式。这将重置对 xref:/use/views/k8s-filters.adoc[视图过滤器] 或 xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[可视化设置] 所做的所有未保存更改。

| `SHIFT` + `1`
| 在视图、子视图或探索模式中，打开 xref:/use/views/k8s-topology-perspective.adoc[拓扑视角]。

| `SHIFT` + `2`
| 在视图、子视图或探索模式中，打开 xref:/use/views/k8s-events-perspective.adoc[事件视角]。

| `SHIFT` + `3`
| 在视图、子视图或探索模式中，打开 xref:/use/views/k8s-metrics-perspective.adoc[指标视角]。

| `SHIFT` + `4`
| 在视图、子视图或探索模式中，打开 xref:/use/views/k8s-traces-perspective.adoc[追踪视角]。
|===
