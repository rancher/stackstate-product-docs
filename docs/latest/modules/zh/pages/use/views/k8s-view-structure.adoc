== 概述

在 SUSE Observability 中，视图允许您监控和检查 IT 环境的一个子集。视图的结构旨在以高效的方式过滤和可视化该子集（视图）中的数据。

=== 过滤器

视图 UI 右上角的 *过滤器* 菜单允许您过滤在视图中显示的组件（拓扑）、事件和追踪。一旦应用，过滤器将影响视图中所有视角的内容。

* xref:/use/views/k8s-filters.adoc[过滤器] - 过滤您视图中的组件（拓扑）、事件和追踪

=== 视角

视图的 *视角* 作为选项卡显示在视图 UI 的左上角，允许您通过不同的视角可视化视图中的所有数据：

* xref:/use/views/k8s-overview-perspective.adoc[概览视角] 或 xref:/use/views/k8s-highlights-perspective.adoc[亮点视角] - 取决于您所处的视图类型
* xref:/use/views/k8s-topology-perspective.adoc[拓扑视角] - 视图组件的依赖关系图
* xref:/use/views/k8s-events-perspective.adoc[事件视角] - 拓扑上发生的所有事件
* xref:/use/views/k8s-metrics-perspective.adoc[指标视角] - 最相关组件的关键指标
* xref:/use/views/k8s-traces-perspective.adoc[追踪视角] - 在拓扑上运行的追踪信息

[NOTE]
====
*所有视角* 将根据 xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc[时间线] 配置更新其内容。
====
