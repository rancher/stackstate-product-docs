指标视角显示所选资源的指标。

image::k8s/k8s-metrics-perspective.png[指标视角]

== 图表

图表显示所选组件的指标数据，近实时获取——数据每30秒获取一次。如果一个处理已停止且不再接收数据，该处理最终会从图表上消失，因为数据至少每30秒向左移动一次。如果在30秒的间隔内有更多数据到达，它将被推送到图表中。
