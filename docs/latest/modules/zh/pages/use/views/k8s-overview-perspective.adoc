= 概述视角
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

概述视角显示您的视图中所有组件的列表。根据视图类型和视图中的组件，概述视角的结构会有所不同。

例如，xref:/use/views/k8s-views.adoc[kubernetes 视图]中使用的表格结构将反映每个视图中包含的组件类型的最重要属性：如下面所示，`services` 视图的表格结构与 `pods` 视图不同。

image::k8s/k8s-overview-perspective-table-comparison.png[概述表格结构比较]

对于 xref:/use/views/k8s-custom-views.adoc[自定义视图] 和 xref:/use/views/k8s-explore-views.adoc[探索视图]，概述视角表格将采用一种通用结构，由最常见的属性构成，因为视图中可能包含多种组件类型。

image::k8s/k8s-overview-perspective-generic-table.png[概述表格通用结构]

尽管在大多数视图类型中，概述视角将采用表格布局，但在某些情况下也会提供卡片布局，允许您在不同的显示模式之间切换概述视角的内容。

image::k8s/k8s-overview-perspective-cards-layout.png[概述卡片布局]
