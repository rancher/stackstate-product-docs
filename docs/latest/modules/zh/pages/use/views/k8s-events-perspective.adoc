== 概述

事件视角显示当前 xref:/use/views/k8s-view-structure.adoc[视图] 或经过过滤后的拓扑中元素的事件和变化。

image::k8s/k8s-events-perspective.png[事件视角]

== 过滤显示的事件

显示的事件可以在视图 UI 右上角的 *过滤器* 菜单中进行过滤。事件过滤器将应用于视图中显示的所有事件：事件视角、高亮视角和时间线。

=== 按属性过滤

在 *过滤器* 菜单中设置的 xref:/use/views/k8s-filters.adoc#_filter_events[事件过滤器] 可用于根据事件类别、类型、源系统和标签来细化显示的事件。

=== 按源组件过滤

在 *过滤器* 菜单中设置的 xref:/use/views/k8s-filters.adoc#_filter_topology[拓扑过滤器] 定义将显示事件的元素（组件和关系）。仅与匹配应用的拓扑过滤器或视图本身的元素相关的事件将可见。您可以通过更新拓扑过滤器来调整显示事件的组件。

=== 按时间戳过滤

事件视角和事件列表显示与视图 UI 底部时间线中选择的 xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_telemetry_interval[遥测间隔] 匹配的事件。调整遥测间隔以仅显示在该时间生成的事件。

== 时间旅行

事件视角列出生成的事件：

* 由在 xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_topology_time[拓扑时间] 指定时刻存在的拓扑元素生成的事件。
* 在选定的 xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_telemetry_interval[遥测间隔] 内

这使您能够穿越时间，既查看由特定历史时刻可用的拓扑元素生成的事件，也查看在特定时间窗口内生成的事件。选择一个新的拓扑时间将使您穿越到历史上该时刻可用的拓扑。事件列表将更新，以包括在该时刻可用的拓扑元素和所选遥测间隔内的事件。

例如：

* 调整*遥测间隔*以增加或减少显示的事件数量。
* 调整*拓扑时间*以穿越到历史上该时刻可用的拓扑。由在 拓扑时间 存在的拓扑元素生成的事件将在所选遥测间隔内显示。
* 单击事件的*属性*中的时间戳以跳转到此拓扑时间。这将更新事件列表，以显示生成的事件：
 ** 由在该时刻存在的拓扑元素生成的事件。
 ** 在指定的遥测间隔内（如果需要，这将调整以适应所选时间戳）。
