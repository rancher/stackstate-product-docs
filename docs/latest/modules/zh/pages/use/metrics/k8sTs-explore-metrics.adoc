= 探索指标
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

您可以在 SUSE Observability 主菜单的底部找到指标探索器。使用它执行任何 PromQL 查询并可视化结果时间序列。查询结果以图表形式显示在所选时间范围内，并在表格中显示最后一个值及时间序列的标签。

image::k8s/k8s-metrics-explorer.png[指标探索器]

== PromQL 查询

ifdef::ss-ff-stackpacks2_enabled[]
查询输入字段具有指标名称、标签名称和值的自动建议，以及支持的 PromQL 函数。请参阅 Prometheus 文档以获取完整的 https://prometheus.io/docs/prometheus/latest/querying/basics/[PromQL 指南和参考]。SUSE Observability 还添加了 2 个默认参数，可在任何查询中使用：`${__interval}` 和 `${__rate_interval}`。它们可以与图表分辨率自动缩放聚合间隔（xref:/setup/custom-integrations/metric-bindings/writing-promql.adoc[更多细节]）。
endif::[]

ifndef::ss-ff-stackpacks2_enabled[]
查询输入字段具有指标名称、标签名称和值的自动建议，以及支持的 PromQL 函数。请参阅 Prometheus 文档以获取完整的 https://prometheus.io/docs/prometheus/latest/querying/basics/[PromQL 指南和参考]。SUSE Observability 还添加了 2 个默认参数，可在任何查询中使用：`${__interval}` 和 `${__rate_interval}`。它们可以与图表分辨率自动缩放聚合间隔（xref:/use/metrics/k8s-writing-promql-for-charts.adoc[更多细节]）。
endif::[]
