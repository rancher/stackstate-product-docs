= 通知
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

在 SUSE Observability 中，当一个 xref:/use/alerting/k8s-monitors.adoc[监控器] 触发时，通知机制会向第三方应用程序发送消息。通知与第三方应用程序集成，如事件管理工具（如PagerDuty或Opsgenie）、工单系统（如JIRA、ServiceNow）或其他协作平台（如Slack）。其他工具可能将通知称为警报。

通知由 SUSE Observability 组件中监控器的健康状态变化触发。通过配置的通知渠道，健康、组件和监控器信息被发送到这些外部系统。
