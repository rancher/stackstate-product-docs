== 排列和调整小部件大小

您可以在仪表板上组织小部件，以创建适合您的布局。

* 要移动小部件，请单击并拖动小部件的标题区域到新位置。
* 要调整小部件的大小，请单击并拖动小部件的右下角。

当您移动或调整小部件的大小时，仪表板上的其他小部件会自动重新排列。

== 编辑或删除小部件

. 要编辑或删除小部件，请单击小部件右上角的更多操作图标。

. 选择 *编辑小部件* 或 *删除小部件*。

要更详细地检查小部件的数据，请单击指标检查器抽屉中“更多操作”按钮旁边的图标。
