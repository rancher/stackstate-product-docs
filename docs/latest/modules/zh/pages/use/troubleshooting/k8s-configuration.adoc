:description: SUSE Observability

如果一个 pod 正在经历崩溃、启动故障或配置错误等问题，检查 YAML 配置可以帮助您识别根本原因。您可以通过点击 *显示配置* 按钮查看 YAML 配置。

image::k8s/k8s-configuration.png[]

这与应用于服务、pod 或其他资源的 `kubectl describe` 命令产生相同的输出。

image::k8s/k8s-configuration-opened.png[]
