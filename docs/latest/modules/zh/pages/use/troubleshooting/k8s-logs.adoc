== 概述

Kubernetes 保持对集群活动和应用程序输出的全面日志记录。日志记录提供了有关容器、节点和 Kubernetes 本身的性能的宝贵见解。从 Pods 到 ReplicaSets 进行细致的日志记录，使您能够追溯问题的根源。

== 访问 Pod 日志

日志中的错误和警告表明，您的应用程序、容器或 Pod 本身可能存在潜在问题。在 Pod 界面上点击 *显示日志* 按钮以查看日志。

== 使用 Pod 日志

您可以使用搜索功能查找日志中的特定错误或警告。日志可以按容器进行过滤。您可以选择希望显示日志的时间间隔以及希望日志更新的刷新间隔。

日志直方图展示选定时间间隔内日志行的数量，并允许您通过点击和拖动将视图缩放至特定时间段。

使用这些选项来显示或隐藏每条日志行的附加信息：

* *显示时间* 以显示时间戳
* *显示容器* 以显示日志行所属的容器名称
* *换行* 以启用长行的换行
* *最新在前/最旧在前* 以切换日志排序顺序
