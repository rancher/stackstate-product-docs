== 概述

使用xref:use/security/k8s-service-tokens.adoc[服务词元]进行身份验证，以便在没有关联的用户帐户的情况下对SUSE Observability进行身份验证。服务词元是用于 CI/CD 身份验证的首选方式，或者在代理或 OpenTelemetry 发送遥测数据时的身份验证方法。如何创建和使用服务词元，在xref:use/security/k8s-service-tokens.adoc[服务词元文档]中有详细说明。
