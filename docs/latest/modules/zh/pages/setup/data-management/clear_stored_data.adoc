= 清除存储的数据
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability 自托管

SUSE Observability 中的数据分为四个不同的集合：

* Elasticsearch 数据
* Kafka 主题数据
* StackGraph data
* 指标数据

在存储如此多的数据时，管理这些数据的手段非常重要。SUSE Observability 中设置了标准的 30 天数据保留期。可以使用 SUSE Observability CLI 根据您的需求进行配置。了解更多关于 xref:/setup/data-management/data_retention.adoc[SUSE Observability 数据保留] 的信息。

== 手动清除数据

要清除在 Kubernetes 上运行的 SUSE Observability 中的存储数据，建议执行干净安装：

. xref:/setup/install-stackstate/kubernetes_openshift/uninstall.adoc#_un_install_the_helm_chart[卸载 SUSE Observability]
. xref:/setup/install-stackstate/kubernetes_openshift/uninstall.adoc#_remove_remaining_resources[去除所有 PVC]
. 再次使用与之前相同的配置在 xref:/setup/install-stackstate/kubernetes_openshift/kubernetes_install.adoc#_deploy_suse_observability_with_helm[Kubernetes] 或 xref:/setup/install-stackstate/kubernetes_openshift/openshift_install.adoc#_deploy_suse_observability_with_helm[OpenShift] 上安装 SUSE Observability。
