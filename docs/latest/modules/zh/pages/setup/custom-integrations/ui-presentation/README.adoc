本节记录了作为 *StackPacks 2.0* 一部分引入的 *用户界面演示* 模型。

== 概述

此处记录的组件演示模型声明了组件（例如，Pods、虚拟机、服务）在 {stackstate-product-name} 用户界面中的呈现方式。
它专注于 *展示相关事项*，例如命名、图标、表格、概述与高亮页面、过滤器，以及菜单项的放置。

该模型的目标是提供一种更可组合和可扩展的方式来定义组件的用户界面行为，相较于现有的、更为分散的方法。
