= README
:page-languages: [en, de, es, fr, ja, pt, zh]
:description: SUSE Observability

OpenTelemetry 提供了 https://opentelemetry.io/docs/languages/[多种语言的 SDK] 来对您的应用程序进行（自动）仪器化。仪器化使您的应用程序能够将跟踪、指标和日志发送到 SUSE Observability。自动仪器化可以捕获流行库和框架的跟踪、指标和日志，而无需进一步的手动干预，只需在您的应用程序中包含 SDK。

OpenTelemetry 的 SDK 数量远超此处文档所列。如果您的语言未在此处文档中列出，您可以使用 https://opentelemetry.io/docs/languages/[opentelemetry.io] 上的文档，并结合 SUSE Observability 所需的 xref:/setup/otel/languages/sdk-exporter-config.adoc[SDK 导出配置]。
