== 禁用处理代理中特定网络协议的跟踪

要禁用处理代理中特定网络协议的跟踪，请配置 Helm 图表设置 `processAgent.disabledProtocols`。以下示例禁用 `http2` 和 `postgres` 协议：

----
--set "processAgent.disabledProtocols[0]=http2" \
--set "processAgent.disabledProtocols[1]=postgres"
----

== 禁用处理代理

如果您的环境不支持处理代理，请通过 Helm 图表设置 `nodeAgent.containers.processAgent.enabled` 禁用它：

----
--set nodeAgent.containers.processAgent.enabled=false
----
