= 与 Rancher 和 Kubernetes 的兼容性
:page-languages: [en, de, es, fr, ja, pt, zh]

{stackstate-product-name} 2.6.3 在运行 RKE2/Kubernetes 的 Rancher 管理的 Kubernetes 集群上受支持。

下表列出了兼容的 Rancher 版本及其对应的支持的 RKE2/Kubernetes 版本。

|===
| Rancher 版本 | RKE2 / Kubernetes Versions

| 2.11.x
| v1.30.11+rke2r1

| 2.12.x
| v1.30.11+rke2r1

| 2.13.x
a|
v1.30.11+rke2r1

v1.31.13+rke2r1

v1.32.10+rke2r1

|===

{stackstate-product-name} 可以使用 SUSE 提供的 Helm 图表在 Kubernetes 或 OpenShift 集群上进行安装。安装需要 Helm v3.x，图表支持以下平台：

* *Kubernetes：*1.25 到 1.33
* *OpenShift：*4.14 到 4.19
