== 默认用户名和密码

[tabs]
====
Kubernetes::
+
--
SUSE Observability 默认配置了以下管理员账户：

* *用户名：* `admin`
* *口令：*在安装期间设置。此信息由 `generate_values.sh` 脚本收集，并以 MD5 哈希格式存储在 `values.yaml` 中。
--

OpenShift::
+
--
SUSE Observability 默认配置了以下管理员账户：

* *用户名：* `admin`
* *口令：*在安装期间设置。此信息由 `generate_values.sh` 脚本收集，并以 MD5 哈希格式存储在 `values.yaml` 中。
--
====
