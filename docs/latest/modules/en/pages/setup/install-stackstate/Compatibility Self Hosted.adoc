= Compatibility with Rancher and Kubernetes

{stackstate-product-name} 2.6.3 is supported on Rancher-managed Kubernetes clusters running RKE2/Kubernetes.

The matrix below lists the compatible Rancher versions and their corresponding supported RKE2/Kubernetes versions.

|===
| Rancher Version | RKE2 / Kubernetes Versions

| 2.11.x
| v1.30.11+rke2r1

| 2.12.x
| v1.30.11+rke2r1

| 2.13.x
a|
v1.30.11+rke2r1

v1.31.13+rke2r1

v1.32.10+rke2r1

|===

{stackstate-product-name} can be installed on a Kubernetes or OpenShift cluster using the Helm charts provided by SUSE Observability. These Helm charts require Helm v3.x to install and are supported on:

* *Kubernetes:* 1.21 to 1.32
* *OpenShift:* 4.9 to 4.17
