= Compatibility with Rancher and Kubernetes

{stackstate-product-name} 2.6.3 is supported on Rancher-managed Kubernetes clusters running RKE2/Kubernetes.

The matrix below lists the compatible Rancher versions and their corresponding supported RKE2/Kubernetes versions.

|===
| Rancher Version | RKE2 / Kubernetes Versions

| 2.11.x
| v1.30.11+rke2r1

| 2.12.x
| v1.30.11+rke2r1

| 2.13.x
a|
v1.30.11+rke2r1

v1.31.13+rke2r1

v1.32.10+rke2r1

|===

{stackstate-product-name} can be installed on Kubernetes or OpenShift clusters using the Helm charts provided by SUSE. Installation requires Helm v3.x, and the charts are supported on the following platforms:

* *Kubernetes:* 1.25 to 1.33
* *OpenShift:* 4.14 to 4.19
