== Security Updates

Security: 0 Open Affected CVEs at Release Time

At the time of this fully tested SUSE Observability release, our VEX-aware vulnerability evaluation reports 0 open affected CVEs across Low, Medium, High, and Critical severities. This assessment takes into account the reviewed vulnerability context published in the SUSE Observability VEX Hub, including CVEs that are not exploitable in SUSE Observability as shipped.

This is a point-in-time assessment for this primary tested release. New CVEs may be published after release, and the reported count may change over time as vulnerability databases are updated.

To reduce that window, we now also publish daily pre-releases, available through Helm with the --devel flag. These development releases let customers and teams pick up rebuilds, dependency updates, and CVE remediation work earlier between the primary tested releases.

Release notes continue to apply to the fully baked release stream, which is published roughly three times every two months. Our goal is for each primary tested release to return to a VEX-aware 0 open affected CVE posture at the moment it ships.

== New Features & Enhancements

* *Notification Toast Messages:* Disabled notification toast messages within the user interface to reduce alert noise.

== Bug Fixes

* *Topology and Overview Filter Resolution:* ** Fixed an issue where topology filters failed to display on the Overview and Topology perspectives for views configured to show them.
** Resolved a bug where configured topology filters were not being properly applied to the data.
** Fixed an issue where overview filters were incorrectly applied to views where they did not belong.
* *OpenMetrics Endpoint 500 Error:* Fixed a bug where the OpenMetrics endpoint of the main `stackstate-server` Docker images returned a `500 Internal Server Error` due to the accidental introduction of duplicate labels.
* *View Loading and Management:* ** Fixed an issue where views missing a specified `viewtype` would fail to load with a `Failed to extract identifier type` error.
** Resolved a UI bug where a deleted view would remain visible in the views list until the browser page was manually refreshed.
* *Platform Log Spam:* Fixed a platform issue where an HBase region server would continuously spam the log files with `Not a regular file` errors.

== Agent Bug Fixes

* No agent-specific bug fixes were reported in this release.
