= Notifications
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Les notifications dans SUSE Observability sont le moyen de notifier les applications tierces lorsqu'un xref:/use/alerting/k8s-monitors.adoc[monitor] se déclenche. Les notifications s'intègrent avec des applications tierces telles que des outils de gestion des incidents (comme PagerDuty ou Opsgenie), des systèmes de billetterie (comme JIRA, ServiceNow) ou d'autres plateformes de collaboration (comme Slack). D'autres outils peuvent désigner les notifications comme des alertes.

Les notifications sont déclenchées par des changements d'état de santé des monitors sur les composants de SUSE Observability. Via les canaux de notification configurés, les informations sur la santé, le composant et le monitor sont envoyées à ces systèmes externes.
