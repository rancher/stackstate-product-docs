== Présentation

La Perspective des événements montre les événements et les changements pour les éléments dans la xref:/use/views/k8s-view-structure.adoc[vue] actuelle ou la topologie filtrée.

image::k8s/k8s-events-perspective.png[Perspective des événements]

== Filtrer les événements affichés

Les événements affichés peuvent être filtrés dans le menu *filtres* en haut à droite de l'interface de la vue. Les filtres d'événements seront appliqués à tous les événements affichés dans la vue : la perspective des événements, les perspectives mises en avant et la chronologie.

=== Filtrer par propriétés

Les xref:/use/views/k8s-filters.adoc#_filter_events[filtres d'événements] définis dans le menu *filtres* peuvent être utilisés pour affiner les événements affichés en fonction de la catégorie d'événement, du type, du système source et des étiquettes.

=== Filtrer par composant source

Les xref:/use/views/k8s-filters.adoc#_filter_topology[filtres de topologie] définis dans le menu *filtres* définissent les éléments (composants et relations) pour lesquels les événements seront affichés. Seuls les événements relatifs aux éléments qui correspondent aux filtres de topologie appliqués ou à la vue elle-même seront visibles. Vous pouvez ajuster les composants pour lesquels les événements sont affichés en mettant à jour les filtres de topologie.

=== Filtrer par horodatage

La Perspective des événements et les listes d'événements montrent les événements qui correspondent à l'xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_telemetry_interval[intervalle de télémétrie] sélectionné dans la chronologie en bas de l'interface de la vue. Ajustez l'intervalle de télémétrie pour afficher uniquement les événements qui ont été générés à ce moment-là.

== Voyage dans le temps

La Perspective des événements liste les événements qui ont été générés :

* par les éléments de topologie qui existaient au moment spécifié par le xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_topology_time[temps de topologie].
* dans l'xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_telemetry_interval[intervalle de télémétrie] sélectionné

Cela vous permet de voyager dans le temps vers les événements générés par les éléments de topologie disponibles à un moment spécifique de l'histoire et les événements générés dans une fenêtre temporelle spécifique. Sélectionner un nouveau temps de topologie vous fera voyager dans le temps vers la topologie qui était disponible à ce moment de l'histoire. La liste des événements sera mise à jour pour inclure les événements des éléments de topologie disponibles à ce moment-là et dans l'intervalle de télémétrie sélectionné.

Par exemple :

* Ajustez l'*intervalle de télémétrie* pour augmenter ou diminuer le nombre d'événements affichés.
* Ajustez le *temps de topologie* pour voyager dans le temps vers la topologie disponible à ce moment de l'histoire. Les événements générés dans l'intervalle de télémétrie sélectionné par les éléments de topologie qui existaient au temps de topologie seront affichés.
* Cliquez sur un horodatage dans les *Propriétés* d'un événement pour sauter à ce temps de topologie. Cela mettra à jour la liste des événements pour afficher les événements qui ont été générés :
 ** par des éléments de topologie qui existaient à ce moment-là.
 ** dans l'intervalle de télémétrie spécifié (cela sera ajusté pour s'adapter à l'horodatage sélectionné si nécessaire).
