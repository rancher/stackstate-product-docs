La perspective des métriques montre les métriques pour la ressource sélectionnée.

image::k8s/k8s-metrics-perspective.png[Perspective des métriques]

== Graphiques

Les graphiques montrent les données métriques pour les composants sélectionnés en temps réel - les données sont récupérées toutes les 30 secondes. Si un processus est interrompu et qu'aucune donnée n'est reçue, le processus finira par quitter le graphique alors que les données se déplacent vers la gauche au moins toutes les 30 secondes. Si d'autres données arrivent pendant l'intervalle de 30 secondes, elles seront ajoutées à un graphique.
