= Explorer les vues
:revdate: 2025-07-10

:description: SUSE Observability

== Présentation

En général, le concept d'une xref:/use/views/k8s-view-structure.adoc[vue] dans SUSE Observability vous permet de surveiller une zone de votre paysage informatique que vous aviez précédemment enregistrée. Mais, souvent, lorsque vous devez enquêter sur un sous-ensemble d'une vue particulière, vous ne voulez pas perdre la portée de cette vue. C'est là que les *vues d'exploration* entrent en jeu.

Pour garder la portée d'une vue particulière (par exemple, `my view`) intacte, toutes les actions d'investigation appliquées à un élément de topologie ou à une sélection d'éléments (par exemple, composants, relations, groupes) s'ouvriront automatiquement dans une vue d'exploration temporaire, sous la vue d'origine (par exemple, `my view / explore`).

Exemples d'*actions d'investigation* qui s'ouvriront automatiquement dans une vue d'exploration :

* Le *bouton +* a été cliqué sur un composant dans la xref:/use/views/k8s-topology-perspective.adoc[perspective de topologie] pour montrer les voisins cachés de ce composant.
* Une *action rapide* a été exécutée sur un composant.
* Un élément de topologie (composant, relation, groupe) a été *double-cliqué* dans la xref:/use/views/k8s-topology-perspective.adoc[perspective de topologie] pour l'examiner.
* Le lien ou le bouton *'Explorer...'* a été cliqué sur un élément de topologie (composant, relation, groupe) pour l'examiner.

[TIP]
====
Les actions d'investigation exécutées à partir d'une vue d'exploration existante *n'ouvriront pas* une nouvelle vue d'exploration - elles changeront la portée de la vue d'exploration dans laquelle vous vous trouvez.
====


== Gestion

Lorsqu'une vue d'exploration est automatiquement créée, sa portée originale est également définie. Ceci est utile si vous souhaitez réinitialiser la vue d'exploration à sa portée originale après avoir apporté des modifications.

Bien qu'une vue d'exploration soit censée être temporaire, elle peut être enregistrée en tant que xref:/use/views/k8s-custom-views.adoc[vue personnalisée] si vous souhaitez la surveiller ou la revoir plus tard.

Si vous ne souhaitez pas enregistrer une vue d'exploration, quittez-la simplement ou revenez à la vue dont vous êtes parti en utilisant les fils d'Ariane dans la barre de navigation supérieure.

== Structure

Les vues d'exploration ont une structure identique à celle des xref:/use/views/k8s-custom-views.adoc[vues personnalisées] : elles ont les mêmes xref:/use/views/k8s-view-structure.adoc#_filters[filtres] et les mêmes xref:/use/views/k8s-view-structure.adoc#_perspectives[perspectives].
