= Perspective d'ensemble
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

La perspective d'ensemble montre une liste de tous les composants dans votre vue. En fonction du type de vue et des composants dans la vue, la structure de la perspective d'ensemble sera différente.

Par exemple, la structure de tableau utilisée dans les xref:/use/views/k8s-views.adoc[vues Kubernetes] reflétera les propriétés les plus importantes des types de composants inclus dans chaque vue : comme on le voit ci-dessous, la vue `services` a une structure de tableau différente de celle de la vue `pods`.

image::k8s/k8s-overview-perspective-table-comparison.png[Comparaison de la structure du tableau d'ensemble]

Pour les xref:/use/views/k8s-custom-views.adoc[vues personnalisées] et les xref:/use/views/k8s-explore-views.adoc[vues d'exploration], le tableau de la perspective d'ensemble aura une structure générique universelle, composée des propriétés les plus courantes, en raison de la diversité des types de composants qui pourraient être inclus dans la vue.

image::k8s/k8s-overview-perspective-generic-table.png[Structure générique du tableau d'ensemble]

Bien qu'une mise en page de tableau soit utilisée dans la plupart des types de vue pour la perspective d'ensemble, dans certains cas, une mise en page de cartes sera également fournie, vous permettant de changer entre différents modes d'affichage du contenu de la perspective d'ensemble.

image::k8s/k8s-overview-perspective-cards-layout.png[Mise en page des cartes d'ensemble]
