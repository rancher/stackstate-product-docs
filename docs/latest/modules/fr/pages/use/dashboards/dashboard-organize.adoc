== Disposer et redimensionner des widgets

Vous pouvez organiser les widgets sur un tableau de bord pour créer une mise en page qui vous convient.

* Pour déplacer un widget, cliquez et faites glisser la zone de titre du widget vers une nouvelle position.
* Pour redimensionner un widget, cliquez et faites glisser le coin inférieur droit du widget.

D'autres widgets sur le tableau de bord se réorganiseront automatiquement lorsque vous déplacerez ou redimensionnerez un widget.

== Modifier ou supprimer un widget

. Pour modifier ou supprimer un widget, cliquez sur l'icône des actions supplémentaires en haut à droite du widget. 

. Sélectionnez *Modifier le widget* ou *Supprimer le widget*.

Pour examiner les données du widget plus en détail, cliquez sur l'icône à côté du bouton « actions supplémentaires » dans le tiroir de l'inspecteur de métriques.
