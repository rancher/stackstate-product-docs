= Configuration YAML
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Si un pod rencontre des problèmes tels que des plantages, des échecs de démarrage ou des erreurs de configuration, l'inspection de la configuration YAML peut vous aider à identifier la cause profonde. Vous pouvez voir la configuration YAML en cliquant sur le bouton *Afficher la configuration*.

image::k8s/k8s-configuration.png[]

Cela donne le même résultat que la commande `kubectl describe` appliquée à un service, un pod ou une autre ressource.

image::k8s/k8s-configuration-opened.png[]
