== Voir la différence depuis le dernier déploiement

Lorsque des problèmes surviennent après avoir appliqué des modifications à vos configurations Kubernetes, examinez le dernier changement de déploiement.

image::k8s/k8s-show-diff.png[]

Comparer la configuration actuelle avec la précédente peut vous aider à identifier les changements spécifiques qui pourraient avoir causé le problème.

image::k8s/k8s-show-diff-opened.png[]
