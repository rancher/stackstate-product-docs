= Raccourcis clavier
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Les raccourcis clavier ci-dessous peuvent être utilisés pour naviguer dans l'interface utilisateur de SUSE Observability.

|===
| Raccourci clavier | Opération

| `CTRL` + `SHIFT` + `F`
| Recherchez un composant par nom dans la xref:/use/views/k8s-topology-perspective.adoc[Perspective Topologie].

| `CTRL` + `S`
| Écraser les paramètres précédemment enregistrés pour une vue. Cela mettra à jour la vue enregistrée pour inclure les modifications apportées aux xref:/use/views/k8s-filters.adoc[filtres de vue] ou aux xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[paramètres de visualisation].

| `CTRL` + `SHIFT` + `S`
| Enregistrez les paramètres de la vue actuelle en tant que nouvelle vue. La nouvelle vue inclura tous les xref:/use/views/k8s-filters.adoc[filtres de vue] et xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[paramètres de visualisation] actuellement appliqués.

| `CTRL` + `SHIFT` + `R`
| Réinitialisez toutes les modifications non enregistrées et revenez à la vue, sous-vue ou mode d'exploration d'origine. Cela réinitialisera toutes les modifications non enregistrées apportées aux xref:/use/views/k8s-filters.adoc[filtres de vue] ou aux xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[paramètres de visualisation].

| `SHIFT` + `1`
| Dans une vue, sous-vue ou mode d'exploration, ouvrez la xref:/use/views/k8s-topology-perspective.adoc[Perspective Topologie].

| `SHIFT` + `2`
| Dans une vue, sous-vue ou mode d'exploration, ouvrez la xref:/use/views/k8s-events-perspective.adoc[Perspective Événements].

| `SHIFT` + `3`
| Dans une vue, sous-vue ou mode d'exploration, ouvrez la xref:/use/views/k8s-metrics-perspective.adoc[Perspective Métriques].

| `SHIFT` + `4`
| Dans une vue, sous-vue ou mode d'exploration, ouvrez la xref:/use/views/k8s-traces-perspective.adoc[Perspective Traces].
|===
