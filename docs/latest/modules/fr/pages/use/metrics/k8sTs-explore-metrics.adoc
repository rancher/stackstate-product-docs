= Explorer les métriques
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Vous pouvez trouver l'explorateur de métriques en bas du menu principal de SUSE Observability. Utilisez-le pour exécuter n'importe quelle requête PromQL et visualiser les séries temporelles résultantes. Le résultat de la requête est affiché dans un graphique pour la plage de temps sélectionnée et dans un tableau qui affiche la dernière valeur ainsi que les étiquettes des séries temporelles.

image::k8s/k8s-metrics-explorer.png[Explorateur de métriques]

== Requêtes PromQL

ifdef::ss-ff-stackpacks2_enabled[]
Le champ de saisie de la requête propose des suggestions automatiques pour les noms de métriques, les noms et valeurs d'étiquettes, ainsi que les fonctions PromQL prises en charge. Consultez la documentation de Prometheus pour un https://prometheus.io/docs/prometheus/latest/querying/basics/[guide et référence PromQL complets]. SUSE Observability ajoute également 2 paramètres par défaut qui peuvent être utilisés dans n'importe quelle requête : `${__interval}` et `${__rate_interval}`. Ils peuvent être utilisés pour ajuster automatiquement l'intervalle d'agrégation en fonction de la résolution du graphique (xref:/setup/custom-integrations/metric-bindings/writing-promql.adoc[plus de détails]).
endif::[]
