== Présentation

Utilisez les xref:use/security/k8s-service-tokens.adoc[tokens de service] pour vous authentifier à SUSE Observability sans avoir de compte utilisateur associé. Les tokens de service sont le moyen privilégié pour s'authentifier depuis CI/CD, ou lors de l'envoi de données de télémétrie depuis des agents ou Open Telemetry. Comment créer et utiliser des tokens de service est décrit dans la xref:use/security/k8s-service-tokens.adoc[documentation des tokens de service].
