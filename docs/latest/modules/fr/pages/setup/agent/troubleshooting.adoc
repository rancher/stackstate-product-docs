== Désactiver le traçage pour un protocole réseau spécifique dans l'agent de traitement

Pour désactiver le traçage pour un protocole réseau spécifique dans l'agent de traitement, configurez le paramètre du Helm chart `processAgent.disabledProtocols`. L'exemple suivant désactive les protocoles `http2` et `postgres` :

----
--set "processAgent.disabledProtocols[0]=http2" \
--set "processAgent.disabledProtocols[1]=postgres"
----

== Désactiver l'agent de traitement

Si l'agent de traitement n'est pas pris en charge dans votre environnement, désactivez-le via le paramètre du Helm chart `nodeAgent.containers.processAgent.enabled` :

----
--set nodeAgent.containers.processAgent.enabled=false
----
