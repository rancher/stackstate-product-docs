= Lisezmoi
:page-languages: [en, de, es, fr, ja, pt, zh]
:description: SUSE Observability

OpenTelemetry fournit https://opentelemetry.io/docs/languages/[SDKs pour de nombreux langages] afin d'(auto-)instrumenter votre application. L'instrumentation permet à votre application d'envoyer des traces, des métriques et des journaux à SUSE Observability. L'auto-instrumentation capture des traces, des métriques et des journaux pour des bibliothèques et des frameworks populaires sans aucune intervention manuelle supplémentaire, sauf l'inclusion du SDK dans votre application.

OpenTelemetry dispose de bien plus de SDKs que ceux documentés ici. Si votre langage n'est pas documenté ici, vous pouvez utiliser la documentation sur https://opentelemetry.io/docs/languages/[opentelemetry.io] en combinaison avec la xref:/setup/otel/instrumentation/sdk-exporter-config.adoc[configuration de l'exportateur SDK] nécessaire pour SUSE Observability.
