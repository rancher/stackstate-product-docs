Cette section documente le modèle *Présentation de l'interface utilisateur* introduit dans le cadre de *StackPacks 2.0*.

== Présentation

Le modèle de présentation des composants documenté ici déclare comment les composants (par exemple, les pods, les VM, les services) sont présentés dans l'interface utilisateur {stackstate-product-name}.
Il se concentre exclusivement sur *les préoccupations de présentation*, telles que les noms, les icônes, les tableaux, les pages de synthèse et de mise en évidence, les filtres et le placement des éléments de menu.

L'objectif de ce modèle est de fournir un moyen plus composable et extensible de définir le comportement de l'interface utilisateur pour les composants, par rapport à l'approche existante, plus fragmentée.
