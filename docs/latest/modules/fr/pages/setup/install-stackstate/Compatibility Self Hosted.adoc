= Compatibilité avec Rancher et Kubernetes

{stackstate-product-name} La version 2.6.3 est prise en charge sur les clusters Kubernetes gérés par Rancher fonctionnant avec RKE2/Kubernetes.

La matrice ci-dessous répertorie les versions de Rancher compatibles et leurs versions RKE2/Kubernetes correspondantes.

|===
| Version de Rancher | Versions RKE2 / Kubernetes

| 2.11.x
| v1.30.11+rke2r1

| 2.12.x
| v1.30.11+rke2r1

| 2.13.x
a|
v1.30.11+rke2r1

v1.31.13+rke2r1

v1.32.10+rke2r1

|===

{stackstate-product-name} peut être installé sur des clusters Kubernetes ou OpenShift en utilisant les charts Helm fournis par SUSE. L'installation nécessite Helm v3.x, et les charts Helm sont pris en charge sur les plateformes suivantes :

* *Kubernetes:* 1.25 à 1.33
* *OpenShift:* 4.14 à 4.19
