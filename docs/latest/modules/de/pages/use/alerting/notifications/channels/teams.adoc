== Teams-Benachrichtigungen konfigurieren

Um Benachrichtigungen an Slack zu senden, befolgen Sie diese Schritte:

. <<_create_a_power_automate_flow,Erstellen Sie einen Power Automate Flow>>
. <<_add_and_test_the_channel,Fügen Sie den Kanal hinzu und testen Sie ihn>>

=== Erstellen Sie einen Power Automate Flow

Erstellen Sie in Teams einen neuen Flow aus der Vorlage "Webhook".

image::k8s/notifications-teams-webhook-template.png[Flow aus der Webhook-Vorlage erstellen]

Wählen Sie das Team und den Kanal aus, in den die Benachrichtigung eingefügt werden soll, und speichern Sie den Flow.

Bearbeiten Sie den Flow und klicken Sie auf das Feld "Wenn eine Teams-Webhook-Anfrage empfangen wird".
Kopieren Sie den HTTP-URL-Parameter.

image::k8s/notifications-teams-select-url.png[URL aus dem Flow auswählen]

== Fügen Sie den Kanal hinzu und testen Sie ihn

image::k8s/configure-teams-channel.png[Teams-Kanal konfigurieren]

Zurück in SUSE Observability können Sie jetzt die Webhook-URL verwenden, um einen Benachrichtigungskanal zu erstellen.

== Teams-Nachrichten für Benachrichtigungen

Wenn eine Benachrichtigung geöffnet oder geschlossen wird, wird eine neue Teams-Nachricht im Kanal erstellt.
