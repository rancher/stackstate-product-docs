= Benachrichtigungen
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Benachrichtigungen in SUSE Observability sind der Weg, Drittanbieteranwendungen zu benachrichtigen, wenn ein xref:/use/alerting/k8s-monitors.adoc[Monitor] in SUSE Observability ausgelöst wird. Benachrichtigungen integrieren sich mit Drittanbieteranwendungen wie Incident-Management-Tools (wie PagerDuty oder Opsgenie), Ticketing-Systemen (wie JIRA, ServiceNow) oder anderen Kollaborationsplattformen (wie Slack). Andere Tools könnten Benachrichtigungen als Alarme bezeichnen.

Benachrichtigungen werden durch Änderungen des Gesundheitszustands von Monitoren in den Komponenten von SUSE Observability ausgelöst. Über die konfigurierten Benachrichtigungskanäle werden die Gesundheits-, Komponenten- und Monitorinformationen an diese externen Systeme gesendet.
