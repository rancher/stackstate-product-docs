= Metriken erkunden
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Sie finden den Metriken-Explorer am Ende des SUSE Observability Menüs. Verwenden Sie ihn, um eine beliebige PromQL-Abfrage auszuführen und die resultierenden Zeitreihen zu visualisieren. Das Abfrageergebnis wird in einem Diagramm für den ausgewählten Zeitraum und in einer Tabelle angezeigt, die den letzten Wert zusammen mit den Labels für die Zeitreihen zeigt.

image::k8s/k8s-metrics-explorer.png[Metriken-Explorer]

== PromQL-Abfragen

ifdef::ss-ff-stackpacks2_enabled[]
Das Eingabefeld für Abfragen bietet automatische Vorschläge für Metriknamen, Labelnamen und Werte sowie unterstützte PromQL-Funktionen. Siehe die Prometheus-Dokumentation für einen vollständigen https://prometheus.io/docs/prometheus/latest/querying/basics/[PromQL-Leitfaden und Referenz]. SUSE Observability fügt auch 2 Standardparameter hinzu, die in jeder Abfrage verwendet werden können: `${__interval}` und `${__rate_interval}`. Sie können verwendet werden, um das Aggregationsintervall automatisch mit der Diagrammauflösung zu skalieren (xref:/setup/custom-integrations/metric-bindings/writing-promql.adoc[weitere Details]).
endif::[]
