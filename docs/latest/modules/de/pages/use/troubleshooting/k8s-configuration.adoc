= YAML-Konfiguration
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Wenn ein Pod Probleme wie Abstürze, Startfehler oder Fehlkonfigurationen hat, kann die Überprüfung der YAML-Konfiguration Ihnen helfen, die Ursache zu identifizieren. Sie können die YAML-Konfiguration sehen, indem Sie auf die Schaltfläche *Konfiguration anzeigen* klicken.

image::k8s/k8s-configuration.png[]

Dies liefert dieselbe Ausgabe wie der Befehl `kubectl describe`, der für einen Dienst, einen Pod oder eine andere Ressource angewendet wird.

image::k8s/k8s-configuration-opened.png[]
