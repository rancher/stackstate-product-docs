== Siehe Unterschied zur letzten Implementierung

Wenn Probleme nach der Anwendung von Änderungen an Ihren Kubernetes-Konfigurationen auftreten, werfen Sie einen Blick auf die letzte Implementierungsänderung.

image::k8s/k8s-show-diff.png[]

Der Vergleich der aktuellen Konfiguration mit der vorherigen kann Ihnen helfen, die spezifischen Änderungen zu identifizieren, die das Problem verursacht haben könnten.

image::k8s/k8s-show-diff-opened.png[]
