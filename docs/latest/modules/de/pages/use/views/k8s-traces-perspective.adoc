= Spurenperspektive
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Die Spurenperspektive zeigt Spannen für eine Komponente und deren verwandte Metriken.
Dies ermöglicht es Ihnen, die Leistung der Anwendungen in Ihrer IT-Infrastruktur direkt in SUSE Observability zu überwachen.

Klicken Sie auf eine beliebige Spanne in der Liste, um die dazugehörigen Nachkommen zu sehen.
Wenn sie erweitert wird, zeigt die Zeitleiste einer Spanne, wann jeder Nachkomme begonnen und abgeschlossen hat.
Elternspannen in derselben Spur sind standardmäßig zusammengeklappt, können jedoch bei Bedarf erweitert werden. +
Spannen sind je nach ihrem OpenTelemetry ServiceName unterschiedlich gefärbt.
Wenn Sie eine Spur inspizieren und die Liste ihrer Spannen sehen, können Sie auf eine beliebige Spanne klicken, um weitere Details zu sehen.

image::k8s/k8s-traces-perspective.png[Spurenperspektive]

== Spuren filtern

Die Spurfilter ermöglichen es Ihnen, die angezeigten Spuren basierend auf dem Status der Spanne (Fehler, Ok oder Unset), dem Elterntyp (Extern, Intern oder [Root]) und der Dauer zu verfeinern.

Zusätzlich zu diesen Filtern entsprechen die Spuren dem *Zeitfenster*, das im Zeitleistensteuerungselement am unteren Rand der SUSE Observability-Benutzeroberfläche ausgewählt wurde.
Passen Sie das Zeitfenster an, um nur Spuren aus dieser Zeit anzuzeigen.

=== Dauer filtern

Wie im zweiten Screenshot gezeigt, ist es möglich, ein Dauerintervall für die Dauer der Spanne auszuwählen, indem man das Histogramm streicht.
Das Herauszoomen kann erreicht werden, indem alle Filter gelöscht oder der gesamte Bereich gestrichen wird.

== Spannendetails

In SUSE Observability zeigt eine xref:/use/views/k8s-view-structure.adoc[Ansicht] eine Unterauswahl Ihrer IT-Infrastruktur für eine bestimmte Kubernetes-Ressource.
Die Spurenperspektive zeigt die mit der Ressource verbundenen Spannen sowie deren Nachkommen.
Da eine Nachkommen-Spanne von einer anderen Ressource stammen kann, ist es möglich, von den Spannendetails zu ihr zu navigieren.
Der Link "Eltern-Spannen" erweitert die Ansicht, um die vollständige Spur anzuzeigen.

image::k8s/k8s-traces-perspective-span-details.png[Details zu einer Nachkommen-Spanne]

Die beiden obenstehenden Bilder veranschaulichen diese Konzepte, indem sie einen Checkout-Service zeigen, dessen Hauptverantwortung darin besteht, eine Bestellung aus einem Warenkorb zu erstellen.
Sie können ein Beispiel für eine Spur und ihre Spannen für eine Anfrage sehen, um eine Bestellung aufzugeben.
Ein Nachkommen-Spanne wurde hervorgehoben und ihre Details werden angezeigt, einschließlich Links zu Komponenten, die damit verbunden sind.

Das Bürsten einer Spur zoomt in einen bestimmten Zeitabschnitt der Spur.
Die Auswahl kann zurückgesetzt werden, indem Sie später klicken.

== Zeitreise

Wenn Sie die Spuren-Perspektive verwenden, können Sie, genau wie in anderen Perspektiven, entweder im Live-Modus sein oder xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_time_travel[in die Vergangenheit reisen].
