= Metrikperspektive
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Die Metrikperspektive zeigt Metriken für die ausgewählte Ressource an.

image::k8s/k8s-metrics-perspective.png[Metrikperspektive]

== Diagramme

Diagramme zeigen Metrikdaten für die ausgewählten Komponenten in nahezu Echtzeit an - die Daten werden alle 30 Sekunden abgerufen. Wenn ein Prozess beendet wird und keine weiteren Daten empfangen werden, wird der Prozess schließlich das Diagramm verlassen, da sich die Daten mindestens alle 30 Sekunden nach links verschieben. Wenn während des 30-Sekunden-Intervalls weitere Daten eintreffen, werden diese dem Diagramm hinzugefügt.
