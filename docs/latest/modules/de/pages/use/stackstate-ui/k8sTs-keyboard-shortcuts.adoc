= Tastaturverknüpfungen
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Die folgenden Tastenkombinationen können verwendet werden, um die SUSE Observability Benutzeroberfläche zu navigieren.

|===
| Tastenkombination | Aktion

| `CTRL` + `SHIFT` + `F`
| Suchen Sie nach dem Namen einer Komponente in der xref:/use/views/k8s-topology-perspective.adoc[Topologie-Perspektive].

| `CTRL` + `S`
| Überschreiben Sie frühere gespeicherte Einstellungen für eine Ansicht. Dies wird die gespeicherte Ansicht aktualisieren, um alle Änderungen an den xref:/use/views/k8s-filters.adoc[Ansichtsfiltern] oder xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[Visualisierungseinstellungen] einzuschließen.

| `CTRL` + `SHIFT` + `S`
| Speichern Sie die aktuellen Ansichtseinstellungen als neue Ansicht. Die neue Ansicht wird alle derzeit angewendeten xref:/use/views/k8s-filters.adoc[Ansichtsfilter] und xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[Visualisierungseinstellungen] enthalten.

| `CTRL` + `SHIFT` + `R`
| Setzen Sie alle nicht gespeicherten Änderungen zurück und kehren Sie zur ursprünglichen Ansicht, Unteransicht oder zum Erkundungsmodus zurück. Dies wird alle nicht gespeicherten Änderungen an den xref:/use/views/k8s-filters.adoc[Ansichtsfiltern] oder xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[Visualisierungseinstellungen] zurücksetzen.

| `SHIFT` + `1`
| Öffnen Sie in einer Ansicht, Unteransicht oder im Erkundungsmodus die xref:/use/views/k8s-topology-perspective.adoc[Topologie-Perspektive].

| `SHIFT` + `2`
| Öffnen Sie in einer Ansicht, Unteransicht oder im Erkundungsmodus die xref:/use/views/k8s-events-perspective.adoc[Ereignisse-Perspektive].

| `SHIFT` + `3`
| Öffnen Sie in einer Ansicht, Unteransicht oder im Erkundungsmodus die xref:/use/views/k8s-metrics-perspective.adoc[Metriken-Perspektive].

| `SHIFT` + `4`
| Öffnen Sie in einer Ansicht, Unteransicht oder im Erkundungsmodus die xref:/use/views/k8s-traces-perspective.adoc[Spuren-Perspektive].
|===
