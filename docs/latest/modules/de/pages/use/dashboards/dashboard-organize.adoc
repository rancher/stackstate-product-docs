== Widgets anordnen und in der Größe ändern.

Sie können die Widgets auf einem Dashboard organisieren, um ein Layout zu erstellen, das für Sie funktioniert.

* Um ein Widget zu verschieben, klicken und ziehen Sie den Titelbereich des Widgets an eine neue Position.
* Um ein Widget in der Größe zu ändern, klicken und ziehen Sie die untere rechte Ecke des Widgets.

Andere Widgets auf dem Dashboard werden sich automatisch neu anordnen, wenn Sie ein Widget verschieben oder in der Größe ändern.

== Ein Widget bearbeiten oder löschen

. Um ein Widget zu bearbeiten oder zu löschen, klicken Sie auf das Symbol für weitere Aktionen in der oberen rechten Ecke des Widgets. 

. Wählen Sie *Widget bearbeiten* oder *Widget löschen*.

Um die Daten des Widgets genauer zu inspizieren, klicken Sie auf das Symbol neben der Schaltfläche "weitere Aktionen" im Metrikinspektor.
