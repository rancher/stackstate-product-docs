Ein Dashboard visualisiert Daten aus Ihren Ressourcen und kann erstellt und angepasst werden, um Ihren Bedürfnissen gerecht zu werden. Ein Dashboard kann Ressourcen aus verschiedenen Bereichen Ihrer IT-Umgebung enthalten.

== Neues Dashboard erstellen

Um Ihr erstes Dashboard zu erstellen:

. Klicken Sie im Hauptmenü auf **Dashboards**.
. Klicken Sie auf die Schaltfläche **Neues Dashboard hinzufügen** in der Mitte des Bildschirms.

Um ein neues Dashboard zu erstellen:

. Klicken Sie im Hauptmenü auf **Dashboards**.
. Klicken Sie auf die Schaltfläche **Neues Dashboard hinzufügen** oben rechts.
. Füllen Sie die Felder **Name** und **Beschreibung** aus.
. (Optional) Wählen Sie **Öffentlich**, um das Dashboard für alle Benutzer sichtbar zu machen.
. (Optional) Wählen Sie **Favoriten**, um das Dashboard im Hauptmenü für einen schnellen Zugriff hinzuzufügen.
. Klicken Sie auf **Dashboard erstellen**.

Sie werden zum neuen leeren Dashboard navigiert.

image::dashboards/add-new-dashboard-drawer.png[Schublade "Neues Dashboard erstellen"]


== Dashboard-Eigenschaften

Sobald Sie sich auf einem Dashboard befinden, können Sie die Eigenschaften eines Dashboards über das Dropdown-Menü neben der Schaltfläche *Speichern...* in der oberen Navigationsleiste bearbeiten und **Dashboard-Eigenschaften bearbeiten** auswählen. Von hier aus können Sie den Namen und die Beschreibung aktualisieren und festlegen, ob das Dashboard öffentlich oder favorisiert ist.

image::dashboards/edit-dashboard-properties-button.png[Standort der Dashboard-Eigenschaften bearbeiten]

image::dashboards/edit-dashboard-properties-drawer.png[Schublade "Dashboard-Eigenschaften bearbeiten"]

=== Öffentliche Dashboards
Standardmäßig können nur Sie Ihre Dashboards sehen. Wenn Sie ein Dashboard öffentlich machen, wird es für alle Benutzer sichtbar. Öffentliche Dashboards sind für andere einsehbar, aber nicht bearbeitbar.

=== Favorisierte Dashboards
Ein Dashboard zu favorisieren, lässt es im Hauptmenü und beim Hinzufügen von Widgets von anderen Seiten prominenter erscheinen. Favorisieren Sie ein Dashboard, wenn Sie schnellen Zugriff darauf wünschen.

Favorisierte Dashboards sind Ihrem Benutzer zugeordnet. Andere Benutzer haben ihre eigenen favorisierten Dashboards.

Sie können ein Dashboard auch favorisieren oder die Favorisierung aufheben, wo immer Sie das Sternsymbol neben dem Namen eines Dashboards sehen.

== Dashboard-Einstellungen

Die Dashboard-Einstellungen geben Ihnen die Kontrolle darüber, ob der Zeitraum, die Bildwiederholrate und die Variablen im Dashboard gespeichert oder von den Betrachtern anpassbar sind. Klicken Sie auf das Zahnrad-Symbol oben rechts in einem Dashboard, um die Dashboard-Einstellungen zu öffnen.

Wenn Sie eines der Kontrollkästchen unter der Überschrift "Speichern..." aktivieren, zum Beispiel den Zeitraum, wird das Dashboard auf den gewählten Zeitraum fixiert, sobald es gespeichert wurde. Wenn es deaktiviert ist, kann der Betrachter des Dashboards den Zeitraum wählen; die Änderung wird in der URL angezeigt, sodass der Link mit anderen Personen geteilt werden kann, die dann das Dashboard mit demselben Zeitraum anzeigen können.
