= Unterstützte Architekturen des SUSE Observability Agent
:revdate: 2025-07-10
:page-revdate: {revdate}

SUSE Observability ist so konzipiert und getestet, dass es auf bestimmten Hardwarearchitekturen läuft. Installation und Betrieb werden auf diesen Systemen unterstützt:

* `x86_64` (auch bekannt als AMD64 oder Intel 64)
* `arm64` (auch bekannt als AArch64)

Stellen Sie sicher, dass Ihre Zielumgebung diese Anforderungen erfüllt, bevor Sie mit einer Implementierung fortfahren. Die Installation oder Ausführung von SUSE Observability auf anderen Architekturen wird nicht unterstützt.
