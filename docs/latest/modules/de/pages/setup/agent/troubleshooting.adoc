== Deaktivieren Sie das Tracing für ein bestimmtes Netzwerkprotokoll im Prozess-Agent.

Um das Tracing für ein bestimmtes Netzwerkprotokoll im Prozess-Agent zu deaktivieren, konfigurieren Sie die Helm-Chart-Einstellung `processAgent.disabledProtocols`. Das folgende Beispiel deaktiviert die Protokolle `http2` und `postgres`:

----
--set "processAgent.disabledProtocols[0]=http2" \
--set "processAgent.disabledProtocols[1]=postgres"
----

== Deaktivieren Sie den Prozess-Agent.

Wenn der Prozess-Agent in Ihrer Umgebung nicht unterstützt wird, deaktivieren Sie ihn über die Helm-Chart-Einstellung `nodeAgent.containers.processAgent.enabled`:

----
--set nodeAgent.containers.processAgent.enabled=false
----
