== Übersicht

[CAUTION]
====
*Überprüfen Sie die auf dieser Seite bereitgestellten Anweisungen, bevor Sie ein Upgrade durchführen!*

Diese Seite bietet spezifische Anweisungen und Details zu allen erforderlichen manuellen Schritten für das Upgrade auf jede unterstützte Version von SUSE Observability. Jede wesentliche Änderung, die Auswirkungen darauf haben könnte, wie SUSE Observability nach dem Upgrade läuft, wird hier beschrieben, wie z. B. eine Änderung der Speicheranforderungen oder der Konfiguration.

*Lesen und befolgen Sie alle Anweisungen von der Version, die Sie derzeit verwenden, bis zur Version, auf die Sie upgraden werden.*
====
