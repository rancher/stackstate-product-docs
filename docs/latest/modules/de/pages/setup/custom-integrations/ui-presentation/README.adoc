Dieser Abschnitt dokumentiert das *UI-Präsentation*-Modell, das als Teil von *StackPacks 2.0* eingeführt wurde.

== Übersicht

Das hier dokumentierte Komponentenpräsentationsmodell erklärt, wie Komponenten (z. B. Pods, VMs, Dienste) in der {stackstate-product-name} Benutzeroberfläche dargestellt werden.
Es konzentriert sich ausschließlich auf *Präsentationsaspekte*, wie Benennung, Symbole, Tabellen, Übersichts- und Hervorhebungsseiten, Filter und die Platzierung von Menüeinträgen.

Ziel dieses Modells ist es, eine besser komponierbare und erweiterbare Möglichkeit zu bieten, das UI-Verhalten für Komponenten zu definieren, im Vergleich zu dem bestehenden, fragmentierteren Ansatz.
