= README
:page-languages: [en, de, es, fr, ja, pt, zh]
:description: SUSE Observability

OpenTelemetry bietet https://opentelemetry.io/docs/languages/[SDKs für viele Sprachen], um Ihre Anwendung (automatisch) zu instrumentieren. Die Instrumentierung ermöglicht es Ihrer Anwendung, Traces, Metriken und Protokolle an SUSE Observability zu senden. Die automatische Instrumentierung erfasst Traces, Metriken und Protokolle für beliebte Bibliotheken und Frameworks, ohne dass weitere manuelle Eingriffe erforderlich sind, außer das SDK in Ihre Anwendung einzufügen.

OpenTelemetry hat viele weitere SDKs, als hier dokumentiert sind. Wenn Ihre Sprache hier nicht dokumentiert ist, können Sie die Dokumentation unter https://opentelemetry.io/docs/languages/[opentelemetry.io] in Kombination mit der xref:/setup/otel/instrumentation/sdk-exporter-config.adoc[SDK Exporter Konfiguration] verwenden, die für SUSE Observability erforderlich ist.
