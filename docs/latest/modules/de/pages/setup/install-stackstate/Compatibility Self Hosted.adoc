= Kompatibilität mit Rancher und Kubernetes

{stackstate-product-name} 2.6.3 wird auf von Rancher verwalteten Kubernetes-Clustern, die RKE2/Kubernetes ausführen, unterstützt.

Die folgende Matrix listet die kompatiblen Rancher-Versionen und ihre entsprechenden unterstützten RKE2/Kubernetes-Versionen auf.

|===
| Rancher-Version | RKE2 / Kubernetes Versions

| 2.11.x
| v1.30.11+rke2r1

| 2.12.x
| v1.30.11+rke2r1

| 2.13.x
a|
v1.30.11+rke2r1

v1.31.13+rke2r1

v1.32.10+rke2r1

|===

{stackstate-product-name} kann auf Kubernetes- oder OpenShift-Clustern mit den von SUSE bereitgestellten Helm-Charts installiert werden. Die Installation erfordert Helm v3.x, und die Charts werden auf den folgenden Plattformen unterstützt:

* *Kubernetes:* 1.25 bis 1.33
* *OpenShift:* 4.14 bis 4.19
