== Übersicht

Verwenden Sie die xref:use/security/k8s-service-tokens.adoc[Service-Tokens], um sich ohne ein zugehöriges Benutzerkonto bei SUSE Observability zu authentifizieren. Service-Tokens sind die bevorzugte Methode zur Authentifizierung von CI/CD oder beim Senden von Telemetriedaten von Agenten oder Open Telemetry. Wie man Service-Tokens erstellt und verwendet, wird in der xref:use/security/k8s-service-tokens.adoc[Dokumentation zu Service-Tokens] beschrieben.
