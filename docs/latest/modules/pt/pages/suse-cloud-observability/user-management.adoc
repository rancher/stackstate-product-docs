= Gerenciamento de usuários
:description: Gerenciamento de usuários do SUSE Cloud Observability

Os usuários dos inquilinos do SUSE Cloud Observability (instâncias do SUSE Cloud Observability) são gerenciados com https://www.keycloak.org/[Keycloak]. Cada cliente (inquilino) tem um domínio Keycloak dedicado. Um link para o console do Keycloak é enviado na mensagem de boas-vindas quando um usuário é criado.

O SUSE Cloud Observability redireciona os usuários para o Keycloak para autenticação. Espera-se que os usuários sejam membros de um ou mais grupos do Keycloak.

Os grupos do Keycloak predefinidos:

* *realm-admin*: Os membros deste grupo podem fazer login no console do domínio Keycloak e realizar operações de gerenciamento de usuários.
* *stackstate-k8s-troubleshooter*: Os usuários deste grupo são atribuídos ao papel de cliente Keycloak `stackstate-k8s-troubleshooter`, que mapeia para o papel do SUSE Cloud Observability com o mesmo nome. O papel concede permissões regulares do SUSE Cloud Observability.
* *stackstate-k8s-admin*: Os usuários deste grupo são atribuídos ao papel de cliente Keycloak `stackstate-k8s-admin`, que mapeia para o papel do SUSE Cloud Observability com o mesmo nome. O papel concede permissões privilegiadas do SUSE Cloud Observability.

== URL de gerenciamento de usuários

Quando um novo usuário é criado, ele recebe uma mensagem de boas-vindas contendo um link para a página de gerenciamento de usuários do Keycloak. Este link é *exclusivamente para o administrador do inquilino*, que é o primeiro usuário por padrão. O formato da URL é: `https://<keycloak_FQDN>/realms/<your_dedicated_Keycloak_realm>/account`.

== Gerenciamento de usuários

* Faça login no Console de Administração do Keycloak.

image::keycloak_admin_console.png[Console de Administração do Keycloak]

=== Gerenciar usuários

* No menu à esquerda, selecione `Users` na seção `Manage`.

==== Adicionando um novo usuário

Clique no botão `Add user` e insira as informações do usuário, como 'Nome de Usuário', 'Email', 'Primeiro Nome' e 'Último Nome'.

. Deixe o campo `Required users actions` vazio.
. Adicione o usuário aos grupos necessários.
. Clique em `Save`.
+
A mensagem de boas-vindas com o link de inscrição e os links para o inquilino do SUSE Cloud Observability, Keycloak Admin e consoles de Conta são enviados por e-mail ao usuário.

NOTE: Para ativar a conta, que inclui confirmação de e-mail e redefinição de senha, o usuário deve seguir o link de inscrição.

image::keycloak_create_user.png[Keycloak Criar Usuário]

==== Atualizando detalhes do usuário

Para *editar detalhes do usuário*, selecione o usuário clicando no Nome de Usuário.

. Altere os detalhes conforme necessário.
. Defina um ou mais `Required user actions`, por exemplo, para forçar os usuários a atualizar a senha ou configurar senhas de uso único.
. Pressione o botão `Save` quando terminar.

image::keycloak_update_user.png[Atualizar Usuário]

==== Apagando um usuário

. Para *apagar um ou mais usuários*, selecione os usuários necessários e pressione o botão `Delete user`.

=== Participação no Grupo

. Faça login no Console de Administração do Keycloak.
. Na seção `Groups`, procure o grupo que deseja gerenciar.
. Clique no nome do grupo para abrir os detalhes do grupo e vá para a aba `Members`.
. Para adicionar um novo membro ao grupo, pressione o botão `Add Member` e selecione os usuários necessários.
. Para excluir usuários do grupo, selecione os usuários da lista e, em seguida, no menu que está na mesma linha do botão `Add member` marcado como "⋮", selecione `Leave group`.
