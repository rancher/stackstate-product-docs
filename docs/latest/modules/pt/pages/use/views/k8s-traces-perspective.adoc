= Perspectiva de Traces
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

A Perspectiva de Traces mostra spans para um componente e suas métricas relacionadas.
Isso permite que você monitore o desempenho dos aplicativos em sua infraestrutura de TI diretamente no SUSE Observability.

Clique em qualquer span na lista para ver os spans descendentes que pertencem a ele.
Quando expandido, a linha do tempo de um span mostra quando cada descendente começou e foi concluído.
Os spans pais no mesmo trace são recolhidos por padrão, mas podem ser expandidos conforme necessário. +
Os spans são coloridos de forma diferente de acordo com seu OpenTelemetry ServiceName.
Ao inspecionar um trace e ver a lista de seus spans, você pode clicar em qualquer span para ver mais detalhes.

image::k8s/k8s-traces-perspective.png[Perspectiva de Traces]

== Filtrar traces

Os filtros de trace permitem que você refine os traces exibidos com base no status do span (Erro, Ok ou Não Definido), tipo de pai (Externo, Interno ou Raiz) e duração.

Além desses filtros, os traces correspondem à *Janela de Tempo* selecionada no controle de linha do tempo na parte inferior da interface do SUSE Observability.
Ajuste a janela de tempo para mostrar apenas os traces daquele período.

=== Filtrando a duração

Como mostrado na segunda captura de tela, é possível selecionar um intervalo de duração para a Duração do span arrastando o histograma.
Reduzir o zoom pode ser alcançado limpando todos os filtros ou arrastando toda a faixa.

== Detalhes do span

No SUSE Observability, uma xref:/use/views/k8s-view-structure.adoc[visualização] mostra uma sub-seleção de sua infraestrutura de TI para um recurso específico do Kubernetes.
A perspectiva de traces mostra os spans relacionados ao recurso, junto com seus descendentes.
Como um span descendente pode originar de outro recurso, é possível navegar até ele a partir dos detalhes do span.
O link "parent spans" expandirá a visualização para mostrar o trace completo.

image::k8s/k8s-traces-perspective-span-details.png[Detalhes de um span descendente]

As duas imagens acima ilustram esses conceitos mostrando um serviço de checkout cuja principal responsabilidade é criar um pedido a partir de um carrinho.
Você pode ver um exemplo de um trace e seus spans para uma solicitação de fazer um pedido.
Um span descendente foi destacado e seus detalhes são mostrados, incluindo links para componentes que estão relacionados a ele.

Da mesma forma, arrastar um trace ampliará uma seção de tempo específica do trace.
A seleção pode ser redefinida clicando novamente.

== Viagem no tempo

Ao usar a Perspectiva de Traces, assim como em outras perspectivas, você pode estar no modo ao vivo ou xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_time_travel[viajar no tempo para o passado].
