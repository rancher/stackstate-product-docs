A Perspectiva de Métricas mostra métricas para o recurso selecionado.

image::k8s/k8s-metrics-perspective.png[Perspectiva de Métricas]

== Gráficos

Os gráficos mostram dados de métricas para os componentes selecionados em tempo real - os dados são coletados a cada 30 segundos. Se um processo for parado e nenhum dado for mais recebido, o processo eventualmente sairá do gráfico à medida que os dados se deslocam para a esquerda a cada 30 segundos. Se mais dados chegarem durante o intervalo de 30 segundos, eles serão adicionados a um gráfico.
