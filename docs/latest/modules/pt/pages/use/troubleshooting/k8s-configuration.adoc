= Configuração YAML
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Se um pod estiver enfrentando problemas como falhas, falhas ao iniciar ou configurações incorretas, inspecionar a configuração YAML pode ajudá-lo a identificar a causa raiz. Você pode ver a configuração YAML clicando no botão *Mostrar configuração*.

image::k8s/k8s-configuration.png[]

Isso fornece a mesma saída que o comando `kubectl describe` aplicado a serviço, pod ou outro recurso.

image::k8s/k8s-configuration-opened.png[]
