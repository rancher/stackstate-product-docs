== Veja a diferença da última implantação.

Quando surgirem problemas após aplicar alterações nas configurações do Kubernetes, dê uma olhada na última alteração na implantação.

image::k8s/k8s-show-diff.png[]

Comparar a configuração atual com a anterior pode ajudá-lo a identificar as alterações específicas que podem ter causado o problema.
