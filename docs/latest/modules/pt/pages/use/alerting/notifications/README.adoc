= Notificações
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

As notificações no SUSE Observability são a forma de notificar aplicativos de terceiros quando um xref:/use/alerting/k8s-monitors.adoc[monitor] no SUSE Observability é acionado. As notificações se integram a aplicativos de terceiros, como ferramentas de gerenciamento de incidentes (como PagerDuty ou Opsgenie), sistemas de tickets (como JIRA, ServiceNow), ou outras plataformas de colaboração (como Slack). Outras ferramentas podem se referir a notificações como alertas.

As notificações são acionadas por mudanças no estado de saúde dos monitores nos componentes do SUSE Observability. Através dos canais de notificação configurados, as informações de saúde, componente e monitor são enviadas para esses sistemas externos.
