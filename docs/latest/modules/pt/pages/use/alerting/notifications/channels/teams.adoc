== Configurar notificações do Teams

Para enviar notificações para o Slack, siga estas etapas:

. <<_create_a_power_automate_flow,Criar um Fluxo do Power Automate>>
. <<_add_and_test_the_channel,Adicionar e testar o canal>>

=== Criar um Fluxo do Power Automate

No Teams, crie um novo Fluxo a partir do modelo "Webhook".

image::k8s/notifications-teams-webhook-template.png[Criar Fluxo a partir do modelo Webhook]

Selecione a Equipe e o Canal para onde você deseja que a notificação seja enviada e salve o fluxo.

Edite o fluxo e clique na caixa "Quando um pedido de webhook do Teams é recebido".
Copie o parâmetro da URL HTTP.

image::k8s/notifications-teams-select-url.png[Selecionar URL do Fluxo]

== Adicionar e testar o canal

image::k8s/configure-teams-channel.png[Configurar Canal do Teams]

De volta ao SUSE Observability, agora você pode usar a URL do Webhook para criar um canal de notificação.

== Mensagens do Teams para notificações
