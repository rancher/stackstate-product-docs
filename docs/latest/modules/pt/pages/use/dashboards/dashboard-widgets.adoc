= Widgets do painel de controle
:revdate: 2025-11-24
:page-revdate: {revdate}
:description: Visualize os dados dos seus recursos com vários widgets.

Widgets são os blocos de construção do seu painel. Eles visualizam dados, geralmente de um ou mais recursos do Kubernetes. Widgets também podem mostrar informações de texto estático para os visualizadores do painel.

== Adicionando um novo widget

Você pode adicionar um novo widget de duas maneiras:

* Clique no botão *Adicionar Widget* no canto superior direito de um painel.

* Em um painel vazio, clique na seção *Adicionar Widget* no meio da tela. Uma gaveta se abre onde você pode configurar o novo widget.

image::dashboards/add-a-widget.png[Formas diferentes de adicionar um widget]

== Adicionando um widget existente

Ao visualizar um componente individual em {stackstate-product-name}, você pode adicionar gráficos diretamente a um painel usando o menu "mais ações" e selecionando "Adicionar ao painel".

image::dashboards/add-existing-widget.png[Adicionando um widget existente]

Isso também está disponível para quaisquer itens fixados, através do mesmo menu "mais ações" em um widget individual.

Além disso, você pode adicionar todos os itens fixados a um painel, usando as "mais ações" na gaveta de itens fixados ou clicando no botão "Adicionar todos os itens ao painel" na parte inferior da gaveta.

image::dashboards/add-all-pinned-items.png[Adicionando todos os itens fixados]

Os widgets resultantes no painel têm um link para o recurso de onde foram adicionados.

== Configurando um widget

Cada widget tem um número de opções de configuração. Algumas são comuns a todos os widgets, outras são baseadas no tipo de widget.

Para editar a configuração de um widget, use o menu "mais ações" e selecione "Editar widget".

image::dashboards/configure-a-widget.png[Adicionando todos os itens fixados]

=== Campos comuns de widgets

|===
| Nome do campo | Descrição

| Nome
| Um nome curto e descritivo exibido no cabeçalho do widget.

| Descrição
| Detalhes adicionais exibidos no cabeçalho do widget atrás de uma dica de ferramenta.

| Tipo
| O <<types,tipo de visualização>>.
|===

Além desses campos, existem algumas abas comuns a todos os widgets: 

|===
| Nome da aba | Descrição

| Links
| Links de recursos relacionados, exibidos abaixo do nome do widget. O primeiro link é sempre exibido. Outros links estão disponíveis em um menu suspenso.

| YAML
| A configuração YAML bruta para o widget, que pode ser visualizada e editada.
|===


=== Aba de consultas

Os tipos de widget destinados a mostrar dados de recursos terão uma aba *Consultas*. Você pode adicionar uma ou mais consultas PromQL para recuperar os dados para o widget.


=== Tipos [[types]]

==== TimeSeriesChart

O `TimeSeriesChart` visualiza dados de séries temporais em um gráfico de linhas. Cada consulta na configuração do widget produz uma linha no gráfico.

image::dashboards/time-series-chart-example.png[Exemplo de gráfico de séries temporais]

Você pode configurar as seguintes opções:

* *Legenda* - A posição e o modo da legenda.
* **Visual** - Conectar valores nulos para criar uma linha contínua.
* **Y Axis** - A unidade, rótulo e valores mínimo/máximo para o eixo Y.
* **Limiares** - Adicionar linhas horizontais ao gráfico para marcar limiares de valor importantes.

==== BarChart

O `BarChart` visualiza dados de séries temporais em um gráfico de barras. Possui as mesmas configurações para legenda, eixo Y e limiares que o `TimeSeriesChart`.

image::dashboards/bar-chart-example.png[Exemplo de gráfico de barras]

==== Gráfico Estatístico

O `StatChart` mostra um gráfico de sparkline e um único valor que é fácil de ler à primeira vista. Os limiares podem ser configurados para mudar a cor do texto com base no valor.

image::dashboards/gauge-chart-example.png[Exemplo de gráfico de medidor]

Você pode configurar as seguintes opções:

* **Sparkline** - Mostrar ou ocultar o gráfico de sparkline.
* **Unidade** - A unidade do valor.
* **Decimais** - O número de decimais a serem exibidos.
* **Cálculo** - O método usado para calcular o valor a partir dos dados de séries temporais.

==== GaugeChart

O `GaugeChart` exibe um medidor com um valor no meio. A cor do medidor e do texto mudam quando o valor ultrapassa um limiar.

image::dashboards/stat-chart-example.png[Exemplo de gráfico estatístico]

Você pode configurar as seguintes opções:

* **Unidade** - A unidade do valor.
* **Decimais** - O número de decimais a serem exibidos.
* **Máximo** - O valor máximo do medidor.
* **Cálculo** - O método usado para calcular o valor a partir dos dados de séries temporais.

==== Markdown

O `Markdown` widget exibe texto em markdown. Isso pode ser usado para adicionar explicações ou outras informações ao seu painel.

image::dashboards/markdown-chart-example.png[Exemplo de gráfico em markdown]
