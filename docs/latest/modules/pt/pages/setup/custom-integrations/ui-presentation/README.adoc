Esta seção documenta o modelo *Apresentação da interface do usuário* introduzido como parte do *StackPacks 2.0*.

== Visão Geral

O modelo de Apresentação de Componentes documentado aqui declara como os componentes (por exemplo, pods, VMs, serviços) são apresentados na {stackstate-product-name} interface do usuário.
Ele se concentra exclusivamente em *questões de apresentação*, como nomes, ícones, tabelas, páginas de visão geral e destaque, filtros e posicionamento de itens de menu.

O objetivo deste modelo é fornecer uma maneira mais composável e extensível de definir o comportamento da interface do usuário para componentes, em comparação com a abordagem existente, mais fragmentada.
