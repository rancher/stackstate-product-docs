== Desative o rastreamento para um protocolo de rede específico no agente de processo

Para desativar o rastreamento para um protocolo de rede específico no agente de processo, configure a configuração do Helm chart `processAgent.disabledProtocols`. O seguinte exemplo desativa os protocolos `http2` e `postgres`:

----
--set "processAgent.disabledProtocols[0]=http2" \
--set "processAgent.disabledProtocols[1]=postgres"
----

== Desative o agente de processo

Se o agente de processo não for suportado em seu ambiente, desative-o por meio da configuração do Helm chart `nodeAgent.containers.processAgent.enabled`:

----
--set nodeAgent.containers.processAgent.enabled=false
----
