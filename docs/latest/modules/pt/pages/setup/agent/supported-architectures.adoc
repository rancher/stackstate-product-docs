= Arquiteturas suportadas pelo SUSE Observability Agent
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}

O SUSE Observability é projetado e testado para rodar em arquiteturas de hardware específicas. A instalação e operação são suportadas nesses sistemas:

* `x86_64` (também conhecido como AMD64 ou Intel 64)
* `arm64` (também conhecido como AArch64)

Certifique-se de que seu ambiente de destino atenda a esses requisitos antes de prosseguir com uma implantação. Instalar ou executar o SUSE Observability em outras arquiteturas não é suportado.
