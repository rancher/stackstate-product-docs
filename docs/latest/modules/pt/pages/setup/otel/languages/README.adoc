:description: SUSE Observability

O OpenTelemetry fornece https://opentelemetry.io/docs/languages/[SDKs para muitas linguagens] para (auto-)instrumentar seu aplicativo. A instrumentação permite que seu aplicativo envie traços, métricas e logs para o SUSE Observability. A auto-instrumentação captura traços, métricas e logs para bibliotecas e frameworks populares sem qualquer intervenção manual adicional, exceto incluir o SDK em seu aplicativo.

O OpenTelemetry possui muito mais SDKs do que os documentados aqui. Se sua linguagem não estiver documentada aqui, você pode usar a documentação em https://opentelemetry.io/docs/languages/[opentelemetry.io] em combinação com a xref:/setup/otel/languages/sdk-exporter-config.adoc[configuração do exportador SDK] necessária para o SUSE Observability.
