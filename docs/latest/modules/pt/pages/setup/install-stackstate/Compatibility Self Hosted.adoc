= Compatibilidade com Rancher e Kubernetes

{stackstate-product-name} 2.6.3 é suportado em clusters Kubernetes gerenciados pelo Rancher que executam RKE2/Kubernetes.

A matriz abaixo lista as versões compatíveis do Rancher e suas respectivas versões suportadas do RKE2/Kubernetes.

|===
| Versão do Rancher | RKE2 / Kubernetes Versions

| 2.11.x
| v1.30.11+rke2r1

| 2.12.x
| v1.30.11+rke2r1

| 2.13.x
a|
v1.30.11+rke2r1

v1.31.13+rke2r1

v1.32.10+rke2r1

|===

{stackstate-product-name} pode ser instalado em clusters Kubernetes ou OpenShift usando os charts do Helm fornecidos pela SUSE. A instalação requer Helm v3.x, e os charts são suportados nas seguintes plataformas:

* *Kubernetes:* 1.25 a 1.33
* *OpenShift:* 4.14 a 4.19
