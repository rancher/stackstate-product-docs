== Visão Geral

Use os xref:use/security/k8s-service-tokens.adoc[tokens de serviço] para autenticar-se no SUSE Observability sem ter uma conta de usuário associada. Tokens de serviço são a forma preferida de autenticação a partir de CI/CD, ou ao enviar dados de telemetria de agentes ou Open Telemetry. Como criar e usar tokens de serviço está descrito na xref:use/security/k8s-service-tokens.adoc[documentação de tokens de serviço].
