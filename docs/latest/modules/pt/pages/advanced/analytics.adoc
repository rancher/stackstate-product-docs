= Página de Análise
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability Auto-hospedado

A página de Análise fornece uma interface poderosa e flexível para consultar e analisar seus dados de observabilidade usando xref:/develop/reference/k8sTs-stql_reference.adoc[consultas STQL].
Este ambiente interativo capacita você a ir além dos painéis pré-construídos e obter insights mais profundos sobre a saúde, desempenho e dependências do seu ambiente dinâmico.

[CAUTION]
====
*A Página de Análise está descontinuada, mas você ainda pode ativá-la configurando permissões específicas. No entanto, recomendamos fortemente aproveitar os StackPacks para uma observabilidade e análise aprimoradas, pois eles fornecem uma alternativa mais robusta e ativamente suportada.*

Duas permissões são necessárias para ativar a Página de Análise:

* `access-analytics`
* `execute-scripts`

Mais detalhes: xref:/setup/security/rbac/rbac_permissions.adoc[Gerenciamento de permissões]
