= Configuración YAML

:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Si un pod está experimentando problemas como bloqueos, fallos, fallos al iniciar o configuraciones erróneas, inspeccionar la configuración YAML puede ayudarte a identificar la causa raíz. Puedes ver la configuración YAML haciendo clic en el botón *Mostrar configuración*.

image::k8s/k8s-configuration.png[]

Esto da la misma salida que el comando `kubectl describe` aplicado a servicio, pod u otro recurso.
