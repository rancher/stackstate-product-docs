== Consulta la diferencia de la última ampliación

Cuando surjan problemas tras aplicar cambios en tus configuraciones de Kubernetes, echa un vistazo al último cambio en la ampliación.

image::k8s/k8s-show-diff.png[]

Comparar la configuración actual con la anterior puede ayudarte a identificar los cambios específicos que podrían haber causado el problema.
