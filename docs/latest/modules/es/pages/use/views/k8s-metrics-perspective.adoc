La Perspectiva de Métricas muestra métricas para el recurso seleccionado.

image::k8s/k8s-metrics-perspective.png[Perspectiva de Métricas]

== Gráficas

Los gráficos muestran datos de métricas para los componentes seleccionados en casi tiempo real: los datos se obtienen cada 30 segundos. Si un proceso termina y no se reciben más datos, el proceso eventualmente saldrá del gráfico a medida que los datos se desplacen a la izquierda al menos cada 30 segundos. Si llegan más datos durante el intervalo de 30 segundos, se añadirán a un gráfico.
