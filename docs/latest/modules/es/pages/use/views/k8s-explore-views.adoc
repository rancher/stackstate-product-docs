== Descripción general

En general, el concepto de una xref:/use/views/k8s-view-structure.adoc[vista] en SUSE Observability te permite monitorizar un área de tu paisaje de TI que habías guardado previamente. Pero, a menudo, cuando necesitas investigar un subconjunto de una vista particular, no quieres perder el alcance de esa vista. Es aquí donde entran en juego las *vistas de exploración*.

Para mantener el alcance de una vista particular (por ejemplo, `my view`) intacto, todas las acciones de investigación aplicadas a un elemento de topología o selección de elementos (por ejemplo, componentes, relaciones, grupos) se abrirán automáticamente en una vista de exploración temporal, bajo la vista desde la que has comenzado (por ejemplo, `my view / explore`).

Ejemplos de *acciones de investigación* que se abrirán automáticamente en una vista de exploración:

* Se hizo clic en el *botón +* en un componente en la xref:/use/views/k8s-topology-perspective.adoc[perspectiva de topología] para mostrar los vecinos ocultos de ese componente.
* Se ejecutó una *acción rápida* en un componente.
* Se *hizo doble clic* en un elemento de topología (componente, relación, grupo) en la xref:/use/views/k8s-topology-perspective.adoc[perspectiva de topología] para investigarlo.
* Se hizo clic en el enlace o botón *'Explorar...'* en un elemento de topología (componente, relación, grupo) para investigarlo.

[TIP]
====
Las acciones de investigación ejecutadas desde una vista de exploración existente *no* abrirán una nueva vista de exploración; cambiarán el alcance de la vista de exploración en la que te encuentras.
====


== Gestión

Cuando se crea automáticamente una vista de exploración, su alcance original también se define. Esto es útil si deseas restablecer la vista de exploración a su alcance original después de haber realizado algunos cambios en ella.

Aunque una vista de exploración está destinada a ser temporal, se puede guardar como una xref:/use/views/k8s-custom-views.adoc[vista personalizada] si deseas monitorizarla o revisarla más tarde.

Si no deseas guardar una vista de exploración, simplemente aléjate de ella o vuelve a la vista desde la que comenzaste utilizando las migas de pan en la barra de navegación superior.

== Estructura

Las vistas de exploración tienen una estructura idéntica a las xref:/use/views/k8s-custom-views.adoc[vistas personalizadas]: tienen los mismos xref:/use/views/k8s-view-structure.adoc#_filters[filtros] y las mismas xref:/use/views/k8s-view-structure.adoc#_perspectives[perspectivas].
