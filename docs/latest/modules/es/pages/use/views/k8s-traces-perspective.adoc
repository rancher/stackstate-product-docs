= Perspectiva de trazas
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

La Perspectiva de Trazas muestra los spans para un componente y sus métricas relacionadas.
Esto te permite monitorizar el rendimiento de las aplicaciones en tu infraestructura de TI directamente en SUSE Observability.

Haz clic en cualquier span de la lista para ver los spans descendientes que le pertenecen.
Cuando se expande, la línea de tiempo de un span muestra cuándo comenzó y finalizó cada descendiente.
Los spans padres en la misma traza están colapsados por defecto, pero se pueden expandir según sea necesario. +
Los spans están coloreados de manera diferente según su OpenTelemetry ServiceName.
Al inspeccionar una traza y ver la lista de sus spans, puedes hacer clic en cualquier span para ver más detalles.

image::k8s/k8s-traces-perspective.png[Perspectiva de trazas]

== Filtrar trazas

Los filtros de traza te permiten refinar las trazas mostradas según el estado del span (Error, Ok o No establecido), tipo de padre (Externo, Interno o Raíz) y duración.

Además de estos filtros, las trazas coinciden con la *Ventana de Tiempo* seleccionada en el control de línea de tiempo en la parte inferior de la interfaz de SUSE Observability.
Ajusta la ventana de tiempo para mostrar solo trazas de ese tiempo.

=== Filtrando la duración

Como se muestra en la segunda captura de pantalla, es posible seleccionar un intervalo para la Duración del span arrastrando el histograma.
El zoom out se puede conseguir eliminando todos los filtros o arrastrando todo el rango.

== Detalles del span

En SUSE Observability, una xref:/use/views/k8s-view-structure.adoc[vista] te muestra una subselección de tu infraestructura de TI para un recurso particular de Kubernetes.
La perspectiva de trazas muestra los spans relacionados con el recurso, junto con sus descendientes.
Como un span descendiente puede originarse de otro recurso, es posible navegar hacia él desde los detalles del span.
El enlace "spans padres" expandirá la vista para mostrar la traza completa.

image::k8s/k8s-traces-perspective-span-details.png[Detalles de un span descendiente]

Las dos imágenes anteriores ilustran estos conceptos mostrando un servicio de pago cuya principal responsabilidad es crear un pedido a partir de un carrito.
Puedes ver un ejemplo de una traza y sus spans para una solicitud de realizar un pedido.
Se ha destacado un span descendiente y se muestran sus detalles, incluyendo enlaces a componentes que están relacionados con él.

De manera similar, al seleccionar una traza se hará un zoom en una sección temporal particular de la traza.
La selección se puede restablecer haciendo clic posteriormente.

== Viaje en el tiempo

Al utilizar la Perspectiva de Trazas, al igual que en otras perspectivas, puedes estar en modo en vivo o xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_time_travel[viajar en el tiempo al pasado].
