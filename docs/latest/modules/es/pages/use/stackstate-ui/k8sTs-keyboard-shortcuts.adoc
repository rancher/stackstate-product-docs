= Métodos abreviados del teclado
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Los atajos de teclado que aparecen a continuación se pueden utilizar para navegar por la interfaz de SUSE Observability.

|===
| Atajo de teclado | Acción

| `CTRL` + `SHIFT` + `F`
| Busca un componente por nombre en la xref:/use/views/k8s-topology-perspective.adoc[Perspectiva de Topología].

| `CTRL` + `S`
| Sobrescribir la configuración guardada previamente para una vista. Esto actualizará la vista guardada para incluir cualquier cambio realizado en los xref:/use/views/k8s-filters.adoc[filtros de vista] o xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[configuraciones de visualización].

| `CTRL` + `SHIFT` + `S`
| Guarda la configuración de vista actual como una nueva vista. La nueva vista incluirá todos los xref:/use/views/k8s-filters.adoc[filtros de vista] y xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[configuraciones de visualización] aplicados actualmente.

| `CTRL` + `SHIFT` + `R`
| Restablece cualquier cambio no guardado y vuelve a la vista original, subvista o modo de exploración. Esto restablecerá todos los cambios no guardados realizados en los xref:/use/views/k8s-filters.adoc[filtros de vista] o xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[configuraciones de visualización].

| `SHIFT` + `1`
| En una vista, subvista o modo de exploración, abre la xref:/use/views/k8s-topology-perspective.adoc[Perspectiva de Topología].

| `SHIFT` + `2`
| En una vista, subvista o modo de exploración, abre la xref:/use/views/k8s-events-perspective.adoc[Perspectiva de Eventos].

| `SHIFT` + `3`
| En una vista, subvista o modo de exploración, abre la xref:/use/views/k8s-metrics-perspective.adoc[Perspectiva de Métricas].

| `SHIFT` + `4`
| En una vista, subvista o modo de exploración, abre la xref:/use/views/k8s-traces-perspective.adoc[Perspectiva de Trazas].
|===
