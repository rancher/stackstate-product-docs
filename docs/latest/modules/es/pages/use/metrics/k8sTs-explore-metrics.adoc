= Explorar métricas
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Puedes encontrar el explorador de métricas en la parte inferior del menú principal de SUSE Observability. Utilízalo para ejecutar cualquier consulta PromQL y visualizar la serie temporal resultante. El resultado de la consulta se muestra en un gráfico para el rango de tiempo seleccionado y en una tabla que muestra el último valor junto con las etiquetas de la serie temporal.

image::k8s/k8s-metrics-explorer.png[Explorador de métricas]

== Consultas PromQL

ifdef::ss-ff-stackpacks2_enabled[]
El campo de entrada de la consulta tiene autocompletado para nombres de métricas, nombres y valores de etiquetas, y funciones PromQL soportadas. Consulta la documentación de Prometheus para una https://prometheus.io/docs/prometheus/latest/querying/basics/[guía y referencia completa de PromQL]. SUSE Observability también añade 2 parámetros predeterminados que se pueden utilizar en cualquier consulta: `${__interval}` y `${__rate_interval}`. Se pueden utilizar para escalar automáticamente el intervalo de agregación con la resolución del gráfico (xref:/setup/custom-integrations/metric-bindings/writing-promql.adoc[más detalles]).
endif::[]
