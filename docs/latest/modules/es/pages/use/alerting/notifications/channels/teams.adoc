== Configurar notificaciones de Teams

Para enviar notificaciones a Slack, sigue estos pasos:

. <<_create_a_power_automate_flow,Crear un flujo de Power Automate>>
. <<_add_and_test_the_channel,Añadir y probar el canal>>

=== Crear un flujo de Power Automate

En Teams, crea un nuevo flujo a partir de la plantilla "Webhook".

image::k8s/notifications-teams-webhook-template.png[Crear flujo a partir de la plantilla Webhook]

Selecciona el equipo y el canal al que deseas que se pegue la notificación y guarda el flujo.

Edita el flujo y haz clic en la casilla "Cuando se recibe una solicitud de webhook de Teams".
Copia el parámetro de URL HTTP.

image::k8s/notifications-teams-select-url.png[Seleccionar URL del flujo]

== Añadir y probar el canal

image::k8s/configure-teams-channel.png[Configurar canal de Teams]

De vuelta en SUSE Observability, ahora puedes usar la URL del webhook para crear un canal de notificación.
