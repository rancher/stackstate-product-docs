= Notificaciones
:page-languages: [en, de, es, fr, ja, pt, zh]
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Las notificaciones en SUSE Observability son la forma de notificar a aplicaciones de terceros cuando un xref:/use/alerting/k8s-monitors.adoc[monitor] en SUSE Observability se activa. Las notificaciones se integran con aplicaciones de terceros como herramientas de gestión de incidentes (como PagerDuty u Opsgenie), sistemas de tickets (como JIRA, ServiceNow) u otras plataformas de colaboración (como Slack). Otras herramientas pueden referirse a las notificaciones como alertas.

Las notificaciones se activan por cambios en el estado de salud de los monitores en los componentes de SUSE Observability. A través de los canales de notificación configurados, la información de salud, componente y monitor se envía a estos sistemas externos.
