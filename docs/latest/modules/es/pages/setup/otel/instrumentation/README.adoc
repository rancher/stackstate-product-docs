= README
:page-languages: [en, de, es, fr, ja, pt, zh]
:description: SUSE Observability

Open telemetry proporciona https://opentelemetry.io/docs/languages/[SDKs para muchos lenguajes] para (auto-)instrumentar tu aplicación. La instrumentación permite que tu aplicación envíe trazas, métricas y registros a SUSE Observability. La auto-instrumentación captura trazas, métricas y registros para bibliotecas y marcos populares sin necesidad de intervención manual adicional, excepto incluir el SDK en tu aplicación.

Open Telemetry tiene muchos más SDKs de los que están documentados aquí. Si tu lenguaje no está documentado aquí, puedes utilizar la documentación en https://opentelemetry.io/docs/languages/[opentelemetry.io] en combinación con la xref:/setup/otel/instrumentation/sdk-exporter-config.adoc[configuración del exportador de SDK] necesaria para SUSE Observability.
