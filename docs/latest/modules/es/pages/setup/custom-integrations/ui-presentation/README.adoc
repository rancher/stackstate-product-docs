Esta sección documenta el modelo *Presentación de la interfaz de usuario* introducido como parte de *StackPacks 2.0*.

== Descripción general

El modelo de Presentación de Componentes documentado aquí declara cómo se presentan los componentes (por ejemplo, pods, VMs, servicios) en la {stackstate-product-name} interfaz de usuario.
Se centra exclusivamente en *preocupaciones de presentación*, como nombres, iconos, tablas, páginas de resumen y destacados, filtros y colocación de elementos de menú.

El objetivo de este modelo es proporcionar una forma más componible y extensible de definir el comportamiento de la interfaz de usuario para los componentes, en comparación con el enfoque existente, más fragmentado.
