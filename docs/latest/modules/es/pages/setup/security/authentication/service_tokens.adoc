== Descripción general

Utiliza los xref:use/security/k8s-service-tokens.adoc[tokens de servicio] para autenticarte en SUSE Observability sin tener una cuenta de usuario asociada. Los tokens de servicio son la forma preferida de autenticar desde CI/CD, o al enviar datos de telemetría desde agentes o Open Telemetry. Cómo crear y utilizar tokens de servicio se describe en la xref:use/security/k8s-service-tokens.adoc[documentación de tokens de servicio].
