= Arquitecturas soportadas por el Agente de Observabilidad de SUSE
:revdate: 2025-07-10
:page-revdate: {revdate}

SUSE Observability está diseñada y probada para funcionar en arquitecturas de hardware específicas. La instalación y operación están soportadas en estos sistemas:

* `x86_64` (también conocido como AMD64 o Intel 64)
* `arm64` (también conocido como AArch64)

Asegúrate de que tu entorno objetivo cumpla con estos requisitos antes de proceder con una ampliación. No se soporta la instalación o ejecución de SUSE Observability en otras arquitecturas.
