== Desactivar el seguimiento para un protocolo de red específico en el agente de procesos

Para desactivar el seguimiento para un protocolo de red específico en el agente de procesos, configura la opción del chart de Helm `processAgent.disabledProtocols`. El siguiente ejemplo desactiva los protocolos `http2` y `postgres`:

----
--set "processAgent.disabledProtocols[0]=http2" \
--set "processAgent.disabledProtocols[1]=postgres"
----

== Desactivar el agente de procesos

Si el agente de procesos no es compatible en tu entorno, desactívalo a través de la opción del chart de Helm `nodeAgent.containers.processAgent.enabled`:

----
--set nodeAgent.containers.processAgent.enabled=false
----
