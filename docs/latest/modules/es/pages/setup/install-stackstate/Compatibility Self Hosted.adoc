= Compatibilidad con Rancher y Kubernetes

{stackstate-product-name} 2.6.3 es compatible con clústeres de Kubernetes gestionados por Rancher que ejecutan RKE2/Kubernetes.

La matriz a continuación enumera las versiones de Rancher compatibles y sus correspondientes versiones de RKE2/Kubernetes soportadas.

|===
| Versión de Rancher | Versiones de RKE2 / Kubernetes

| 2.11.x
| v1.30.11+rke2r1

| 2.12.x
| v1.30.11+rke2r1

| 2.13.x
a|
v1.30.11+rke2r1

v1.31.13+rke2r1

v1.32.10+rke2r1

|===

{stackstate-product-name} se puede instalar en clústeres de Kubernetes u OpenShift utilizando los charts de Helm proporcionados por SUSE. La instalación requiere Helm v3.x, y los charts son compatibles con las siguientes plataformas:

* *Kubernetes:* 1.25 a 1.33
* *OpenShift:* 4.14 a 4.19
